- **AI 代理（Agent）生态爆发**：从 Anthropic 的 Claude Code 到 Meta 的 Ranking Engineer Agent，AI 正在从简单的聊天机器人转向能够自主执行复杂任务、优化基础设施并进行科学发现的“代理”架构。
