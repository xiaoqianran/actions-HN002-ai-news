### GhostTrack

**GhostTrack**
GhostTrack 是一款用于追踪位置或手机号码的实用工具，因此该工具可被称为 OSINT（开源情报）或信息收集工具。
GhostTrack is a useful tool to track location or mobile number, so this tool can be called OSINT or also information gathering.

Linux (deb) 安装方法：
Installation on Linux (deb)

工具使用方法：
Usage Tool

**Display on the menu: IP Tracker**
菜单显示：IP 追踪器 (IP Tracker)
在 IP 追踪菜单中，您可以结合 Seeker 工具来获取目标 IP。
On the IP Track menu, you can combo with the seeker tool to get the target IP.
⚡ Install Seeker: [Get Seeker](https://github.com/thewhiteh4t/seeker)
⚡ 安装 Seeker：[获取 Seeker](https://github.com/thewhiteh4t/seeker)

**Display on the menu: Phone Tracker**
菜单显示：手机追踪器 (Phone Tracker)
在此菜单中，您可以搜索目标手机号码的相关信息。
On this menu you can search for information from the target phone number.

**Display on the menu: Username Tracker**
菜单显示：用户名追踪器 (Username Tracker)
在此菜单中，您可以搜索目标在社交媒体上的用户名相关信息。
On this menu you can search for information from the target username on social media.
