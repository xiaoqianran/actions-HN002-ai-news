# Amazon is already offering new OpenAI products on AWS
# 亚马逊已在 AWS 上提供 OpenAI 的新产品

Almost as soon as OpenAI announced that its major investor and cloud partner, Microsoft, no longer has exclusive rights to any of its products, Amazon started gloating. After the revised OpenAI/Microsoft agreement was announced on Monday, Amazon CEO Andy Jassy noted in a tweet that it was a “very interesting announcement.”

就在 OpenAI 宣布其主要投资者兼云合作伙伴微软不再对其任何产品拥有独家权利后不久，亚马逊就开始“炫耀”了。周一修订后的 OpenAI/微软协议公布后，亚马逊首席执行官安迪·贾西（Andy Jassy）在推特上评论称，这是一个“非常有趣的公告”。

That agreement solved OpenAI’s problem of allowing AWS to offer its products, an issue that crystalized after it signed an up-to-$50-billion deal with Amazon. Amazon announced on Tuesday that AWS’s Bedrock service now has OpenAI’s latest models, its code-writing service Codex, and a new product for creating OpenAI-powered AI agents.

该协议解决了 OpenAI 允许 AWS 提供其产品的问题，这一问题在 OpenAI 与亚马逊签署了一项高达 500 亿美元的协议后变得尤为突出。亚马逊周二宣布，AWS 的 Bedrock 服务现已接入 OpenAI 的最新模型、代码编写服务 Codex，以及一款用于创建 OpenAI 驱动的 AI 智能体（AI agents）的新产品。

Bedrock is Amazon’s AI app building and model-choosing service. Amazon is calling the new agent service Bedrock Managed Agents. It is specifically designed to use OpenAI’s reasoning models, offering features like agent steering and security.

Bedrock 是亚马逊的 AI 应用构建和模型选择服务。亚马逊将这项新的智能体服务称为“Bedrock Managed Agents”。它专为使用 OpenAI 的推理模型而设计，提供诸如智能体引导（agent steering）和安全性等功能。

Amazon promises in its blog post that “this is the beginning of a deeper collaboration between AWS and OpenAI.” And it will certainly be interesting to watch. The Microsoft/OpenAI relationship has reportedly been deteriorating for some time, with each of them finding comfort in the arms of their partner’s biggest rival.

亚马逊在博客文章中承诺，“这是 AWS 与 OpenAI 之间更深层次合作的开始。”这无疑值得关注。据报道，微软与 OpenAI 的关系已经恶化了一段时间，双方都在各自合作伙伴的最大竞争对手那里寻求慰藉。

OpenAI has turned to AWS and Oracle. Microsoft to Anthropic; the Redmond-based software giant is also working on a new agent offering powered by Claude.

OpenAI 转向了 AWS 和甲骨文（Oracle），而微软则转向了 Anthropic；这家总部位于雷德蒙德的软件巨头也正在开发一款由 Claude 驱动的全新智能体产品。
