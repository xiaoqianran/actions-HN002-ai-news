### Title: Soft launch of open-source code platform for government
### 标题：政府开源代码平台正式上线（试运行）

The code.overheid.nl (Dutch) platform is now live. It is the government-wide code platform for publishing and developing open-source software. The platform is fully self-hosted and supports digital sovereignty.
code.overheid.nl 平台现已正式上线。这是荷兰政府用于发布和开发开源软件的统一代码平台。该平台完全实现自托管，旨在支持数字主权。

For now, this is a pilot using Forgejo, an open-source, European, and sovereign alternative to GitHub and GitLab. Not all government organisations can use the platform yet. Developers are invited to contribute, with the aim of eventually growing it into a shared Git platform for government bodies.
目前，该平台处于试点阶段，采用的是 Forgejo——一个开源、欧洲本土且具备主权属性的 GitHub 和 GitLab 替代方案。现阶段并非所有政府机构都能使用该平台。我们诚邀开发者参与贡献，目标是最终将其打造为政府部门共享的 Git 平台。

Get involved: The code platform is initiated by the Open Source Program Office at the Ministry of the Interior and Kingdom Relations (BZK), in collaboration with DAWO (SSC-ICT), Opensourcewerken (Dutch), and developer.overheid.nl. Interested in joining? Please email codeplatform@rijksoverheid.nl.
参与方式：该代码平台由荷兰内政与王国关系部 (BZK) 下属的开源项目办公室发起，并与 DAWO (SSC-ICT)、Opensourcewerken 以及 developer.overheid.nl 合作开发。如有意参与，请发送电子邮件至 codeplatform@rijksoverheid.nl。

Further information is available in Dutch, in the blog ‘We gaan samen code.overheid.nl bouwen’ (‘We’re building code.overheid.nl together’) on developer.overheid.nl. Explore the code platform at code.overheid.nl.
更多信息请参阅 developer.overheid.nl 上的荷兰语博客文章《We gaan samen code.overheid.nl bouwen》（意为“我们正在共同构建 code.overheid.nl”）。欢迎访问 code.overheid.nl 探索该代码平台。
