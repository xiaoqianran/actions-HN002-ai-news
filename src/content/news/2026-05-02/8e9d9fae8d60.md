# When Your LLM Reaches End-of-Life: A Framework for Confident Model Migration in Production Systems
# 当你的大语言模型（LLM）生命周期终结时：生产系统中模型迁移的置信框架

**Abstract:** We present a framework for migrating production Large Language Model (LLM) based systems when the underlying model reaches end-of-life or requires replacement. The key contribution is a Bayesian statistical approach that calibrates automated evaluation metrics against human judgments, enabling confident model comparison even with limited manual evaluation data.

**摘要：** 我们提出了一种针对生产环境大语言模型（LLM）系统的迁移框架，旨在应对底层模型达到生命周期终点或需要更换的情况。该研究的核心贡献在于引入了一种贝叶斯统计方法，通过将自动化评估指标与人类判断进行校准，即使在人工评估数据有限的情况下，也能实现高置信度的模型对比。

We demonstrate this framework on a commercial question-answering system serving 5.3M monthly interactions across six global regions; evaluating correctness, refusal behavior, and stylistic adherence to successfully identify suitable replacement models.

我们在一个服务于全球六大区域、每月处理 530 万次交互的商业问答系统上验证了该框架；通过评估准确性、拒绝行为以及风格一致性，成功识别出了合适的替代模型。

The framework is broadly applicable to any enterprise deploying LLM-based products, providing a principled, reproducible methodology for model migration that balances quality assurance with evaluation efficiency. This is a capability increasingly essential as the LLM ecosystem continues to evolve rapidly and organizations manage portfolios of AI-powered services across multiple models, regions, and use cases.

该框架广泛适用于任何部署 LLM 相关产品的企业，为模型迁移提供了一种原则性强且可复现的方法论，在质量保证与评估效率之间取得了平衡。随着 LLM 生态系统的快速演进，以及各组织需要管理跨多个模型、区域和用例的 AI 服务组合，这种能力正变得日益重要。
