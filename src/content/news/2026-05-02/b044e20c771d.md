# Think it, Run it: Autonomous ML pipeline generation via self-healing multi-agent AI
# 构思即运行：通过自愈式多智能体 AI 实现自动化机器学习流水线生成

**Abstract:** The purpose of our paper is to develop a unified multi-agent architecture that automates end-to-end machine learning (ML) pipeline generation from datasets and natural-language (NL) goals, improving efficiency, robustness and explainability.

**摘要：** 本文旨在开发一种统一的多智能体架构，能够根据数据集和自然语言（NL）目标自动生成端到端的机器学习（ML）流水线，从而提高效率、鲁棒性和可解释性。

A five-agent system is proposed to handle profiling, intent parsing, microservice recommendation, Directed Acyclic Graph (DAG) construction and execution. It integrates code-grounded Retrieval-Augmented Generation (RAG) for microservice understanding, an explainable hybrid recommender combining multiple criteria, a self-healing mechanism using Large Language Model (LLM)-based error interpretation and adaptive learning from execution history.

研究提出了一个包含五个智能体的系统，分别负责数据分析、意图解析、微服务推荐、有向无环图（DAG）构建以及执行。该系统集成了基于代码的检索增强生成（RAG）技术以实现对微服务的理解，采用了一种结合多准则的可解释混合推荐器，并利用基于大语言模型（LLM）的错误解读和执行历史自适应学习机制，构建了自愈功能。

The approach is evaluated on 150 ML tasks across diverse scenarios. The system achieves an 84.7% end-to-end pipeline success rate, outperforming baseline methods. It demonstrates improved robustness through self-healing and reduces workflow development time compared to manual construction.

该方法在不同场景下的 150 项机器学习任务中进行了评估。系统实现了 84.7% 的端到端流水线成功率，优于基准方法。实验表明，通过自愈机制，系统的鲁棒性得到了提升，且相比手动构建，显著缩短了工作流的开发时间。

The study introduces a novel integration of code-grounded RAG, explainable recommendation, self-healing execution and adaptive learning within a single architecture, showing that tightly coupled intelligent components can outperform isolated solutions.

本研究在单一架构内创新性地集成了基于代码的 RAG、可解释推荐、自愈式执行和自适应学习，证明了紧密耦合的智能组件能够超越孤立的解决方案。
