# HiveTerm

**HiveTerm: One workspace for all your AI agents**
**HiveTerm：所有 AI 智能体的统一工作空间**

HiveTerm is a desktop terminal that runs your AI agents (Claude Code, Codex, Gemini, Cline, etc.) alongside servers, builds, and scripts, all orchestrated by a hive.yml file.
HiveTerm 是一款桌面终端，它能将你的 AI 智能体（如 Claude Code、Codex、Gemini、Cline 等）与服务器、构建任务和脚本并行运行，并通过一个 `hive.yml` 文件进行统一编排。

What sets it apart is Queen, an integrated MCP server that allows agents to create sub-agents, read output from other processes, and send notifications to one another.
其独特之处在于内置了名为“Queen”的 MCP 服务器，它允许智能体创建子智能体、读取其他进程的输出，并相互发送通知。

Add to that per-process resource monitoring, Git integration with a diff viewer, automatic stack detection, and context-separated workspaces.
此外，它还具备进程级资源监控、集成 Git 及差异对比查看器、自动技术栈检测以及上下文隔离的工作空间等功能。

Running 5 AI agents across 3 projects? Tabs everywhere, no idea what each one is doing. HiveTerm is one terminal workspace where Claude Code, Codex, Gemini and your dev stack run side by side.
同时在 3 个项目中运行 5 个 AI 智能体？标签页到处都是，根本搞不清每个在做什么。HiveTerm 提供了一个统一的终端工作空间，让 Claude Code、Codex、Gemini 和你的开发技术栈能够并肩运行。

Config-driven (hive.yml), per-agent recap in the sidebar, voice input, and an MCP server so agents can spawn and talk to each other.
它采用配置驱动（hive.yml），在侧边栏提供每个智能体的运行摘要，支持语音输入，并配备 MCP 服务器，使智能体能够相互生成并进行对话。

Free to start — up to 3 projects on the free tier. macOS, Windows, Linux.
免费起步——免费版支持最多 3 个项目。适用于 macOS、Windows 和 Linux。
