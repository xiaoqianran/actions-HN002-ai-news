# Unsupervised Electrofacies Classification and Porosity Characterization in the Offshore Keta Basin Using Wireline Logs
# 利用测井数据对加纳近海 Keta 盆地进行无监督岩电相分类与孔隙度表征

**Abstract:** This study presents an unsupervised machine learning workflow for electrofacies analysis in the offshore Keta Basin, Ghana, where core data are scarce. Six standard wireline logs from Well~C were analysed over a depth interval comprising approximately $11{,}195$ samples.

**摘要：** 本研究提出了一种用于加纳近海 Keta 盆地岩电相分析的无监督机器学习工作流程，该地区岩心数据较为匮乏。研究对来自 C 井的六种标准测井数据进行了分析，涵盖了约 11,195 个样本的深度区间。

K-means clustering was applied in multivariate log space, with the clustering structure evaluated using inertia and silhouette diagnostics. Four clusters were identified, supported by an average silhouette coefficient of approximately $0.50$, indicating moderate but meaningful separation.

在多变量测井空间中应用了 K-means 聚类算法，并利用惯性（inertia）和轮廓系数（silhouette diagnostics）对聚类结构进行了评估。研究识别出四个聚类，其平均轮廓系数约为 0.50，表明聚类结果具有中等程度但意义明确的区分度。

The resulting electrofacies exhibit systematic, depth-continuous patterns associated with variations in clay content, porosity, and rock framework properties, forming a geological continuum from shale-dominated to cleaner sandstone-dominated units.

所得的岩电相表现出系统性的、深度连续的模式，这些模式与粘土含量、孔隙度和岩石骨架性质的变化相关，形成了从页岩主导到较纯净砂岩主导的地质连续体。

The results demonstrate that log-only, unsupervised clustering supported by quantitative metrics provides a robust and reproducible framework for subsurface characterisation. The proposed workflow offers a practical tool for early-stage formation evaluation in frontier offshore basins and a foundation for future integrated studies.

研究结果表明，在定量指标支持下，仅利用测井数据进行无监督聚类，可为地下表征提供一种稳健且可重复的框架。所提出的工作流程为前沿近海盆地的早期地层评价提供了一种实用工具，并为未来的综合研究奠定了基础。
