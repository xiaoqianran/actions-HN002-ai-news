# Fast Log-Domain Sinkhorn Optimal Transport with Warp-Level GPU Reductions
# 基于 Warp 级 GPU 归约的快速对数域 Sinkhorn 最优传输

**Abstract:** Entropic regularized optimal transport (OT) via the Sinkhorn algorithm has become a fundamental tool in machine learning, yet existing implementations either suffer from numerical instability for small regularization parameters or incur significant overhead from deep learning frameworks. 

**摘要：** 通过 Sinkhorn 算法实现的熵正则化最优传输（OT）已成为机器学习中的基础工具，然而现有的实现方式要么在正则化参数较小时面临数值不稳定性问题，要么因深度学习框架而产生巨大的计算开销。

We present FastSinkhorn, a lightweight, native CUDA implementation of the log-domain Sinkhorn algorithm that combines warp-level shuffle reductions with shared-memory tiling to achieve high GPU utilization without sacrificing numerical stability. 

我们提出了 FastSinkhorn，这是一种轻量级的原生 CUDA 实现，用于对数域 Sinkhorn 算法。它结合了 Warp 级洗牌归约（Warp-level shuffle reductions）与共享内存分块（Shared-memory tiling），在不牺牲数值稳定性的前提下实现了极高的 GPU 利用率。

Our solver operates entirely in the log-domain, enabling robust computation for regularization parameters as small as epsilon = 10^{-4} where standard-domain methods fail. 

我们的求解器完全在对数域内运行，能够为小至 epsilon = 10^{-4} 的正则化参数提供稳健的计算，而标准域方法在这些参数下往往会失效。

On dense OT problems with n = m = 8192, our implementation achieves 12x speedup over the widely-used POT library and 5.9x speedup over GPU-accelerated PyTorch baselines, while consuming only 256 MB of GPU memory. 

在 n = m = 8192 的稠密 OT 问题上，我们的实现相比广泛使用的 POT 库实现了 12 倍的加速，相比 GPU 加速的 PyTorch 基准测试实现了 5.9 倍的加速，同时仅消耗 256 MB 的 GPU 显存。

We validate our solver on image color transfer, 3D point cloud matching, and convergence analysis, demonstrating that native CUDA kernels with careful numerical treatment provide a practical and efficient foundation for large-scale optimal transport computation.

我们在图像色彩迁移、3D 点云匹配和收敛性分析上验证了我们的求解器，证明了经过精心数值处理的原生 CUDA 核函数为大规模最优传输计算提供了一个实用且高效的基础。
