# Lithos UI: The Neo-Brutalist React Library (100% Free & Open Source)
# Lithos UI：新粗野主义 React 组件库（100% 免费且开源）

Lithos UI is a **production-ready**, Neo-Brutalist React component library designed for high-impact, high-contrast layouts. It is built to prioritize architectural precision and developer velocity.
Lithos UI 是一个**生产就绪**的新粗野主义（Neo-Brutalist）React 组件库，专为高冲击力、高对比度的布局而设计。它的构建旨在优先考虑架构精度和开发速度。

We enforce a strict engineering philosophy: Zero-Gap Architecture: We completely avoid the CSS gap property. Layouts are built using intentional, manual spacing logic for absolute structural integrity.
我们坚持严格的工程理念：零间隙架构（Zero-Gap Architecture）：我们完全避免使用 CSS 的 `gap` 属性。布局通过刻意的、手动定义的间距逻辑构建，以确保绝对的结构完整性。

YIQ Contrast Engine: Accessibility is mathematically guaranteed. Our built-in global engine automatically calculates whether text and borders should be black or white based on any background hex color you pass it.
YIQ 对比度引擎：可访问性在数学上得到了保证。我们内置的全局引擎会根据你传入的任何背景十六进制颜色，自动计算文本和边框应显示为黑色还是白色。

Universal React Architecture: Framework-agnostic and React Compiler ready. Whether you are building with Next.js, Remix, Vite, or Astro, our strictly separated context hooks guarantee flawless integration and perfect Hot Module Replacement.
通用 React 架构：框架无关且支持 React Compiler。无论你是使用 Next.js、Remix、Vite 还是 Astro 进行构建，我们严格分离的上下文钩子（context hooks）都能保证完美的集成和无瑕的模块热替换（HMR）。
