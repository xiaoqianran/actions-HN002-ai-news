# Boundary-Aware Uncertainty Quantification for Wildfire Spread Prediction
# 面向野火蔓延预测的边界感知不确定性量化

**Abstract:** Reliable wildfire spread prediction is vital for risk-aware emergency planning, yet most deep learning models lack principled uncertainty quantification (UQ). Further, for boundary-sensitive cases like wildfire spread, evaluating models with global metrics alone is often insufficient.

**摘要：** 可靠的野火蔓延预测对于风险感知型应急规划至关重要，然而大多数深度学习模型缺乏原则性的不确定性量化（UQ）。此外，对于像野火蔓延这类对边界敏感的案例，仅使用全局指标来评估模型往往是不够的。

To shift the focus of UQ evaluation toward a more operationally relevant approach, the Fire-Centered Evaluation Region (FCER) framework is introduced as a spatially conditioned protocol to characterize UQ within critical fire zones.

为了将不确定性量化评估的重点转向更具实际操作意义的方法，本文引入了“以火为中心的评估区域”（Fire-Centered Evaluation Region, FCER）框架，作为一种空间条件协议，用于刻画关键火灾区域内的不确定性。

Using FCER, an Ensemble is compared against a distilled single-pass student model on the WildfireSpreadTS dataset. The student model demonstrates comparable calibration and complementary uncertainty ranking in boundary-relevant regimes. Code is available at https://github.com/jonasvilhofunk/WildfireUQ-FCER.

通过使用 FCER，研究人员在 WildfireSpreadTS 数据集上对比了集成模型（Ensemble）与经过蒸馏的单次传递学生模型（distilled single-pass student model）。结果表明，学生模型在边界相关区域表现出了相当的校准能力和互补的不确定性排序效果。代码已开源至：https://github.com/jonasvilhofunk/WildfireUQ-FCER。
