# egenioussBench: A New Dataset for Geospatial Visual Localisation
# egenioussBench：用于地理空间视觉定位的新数据集

**Abstract:** We present egenioussBench, a visual localisation benchmark built on geospatial reference data: a city-scale airborne 3D mesh and a CityGML LoD2 model. This pairing reflects deployable mapping assets and supports true scalability beyond traditional SfM-based approaches.

**摘要：** 我们提出了 egenioussBench，这是一个基于地理空间参考数据构建的视觉定位基准：包含城市规模的机载 3D 网格和 CityGML LoD2 模型。这种组合反映了可部署的测绘资产，并支持超越传统基于 SfM（运动恢复结构）方法的真正可扩展性。

The query data comprise smartphone images with centimetre-accurate, map-independent ground truth obtained via PPK and GCP/CP-aided adjustment. From 2,709 images, we derive a non-co-visible subset by estimating the full co-visibility matrix from rendered depth and selecting a maximum independent set; the released data include a test split of 42 non-co-visible images with withheld ground truth and a validation split of 412 sequential images with poses, e.g. for training of pose regressors and self-validation.

查询数据包含智能手机拍摄的图像，其厘米级精度的、与地图无关的地面真值是通过 PPK（后处理动态差分）和 GCP/CP 辅助平差获得的。我们从 2,709 张图像中导出一个非共视子集，方法是通过渲染深度估计完整的共视矩阵并选择最大独立集；发布的数据包括 42 张保留了地面真值的非共视测试集，以及 412 张带有位姿的连续验证集，例如用于位姿回归器的训练和自我验证。

The benchmark features a public leaderboard evaluated with binning metrics at multiple pose-error thresholds alongside global statistics (median, RMSE, outlier ratio), ensuring fair, like-for-like comparison across mesh- and LoD2-based methods. Together, these design choices expose realistic cross-view and cross-domain challenges while providing a rigorous, scalable path for advancing large-scale visual localisation. 该基准测试提供了一个公共排行榜，通过多个位姿误差阈值下的分箱指标以及全局统计数据（中位数、RMSE、异常值比例）进行评估，确保了基于网格和基于 LoD2 的方法之间公平、同等的比较。总之，这些设计选择揭示了现实中的跨视角和跨领域挑战，同时为推进大规模视觉定位提供了一条严谨且可扩展的路径。我们已在相关链接中提供了评估代码和数据。
