# An Introduction to Meshtastic / Meshtastic 简介

Meshtastic® is a project that enables you to use inexpensive LoRa radios as a long-range off-grid communication platform in areas without existing or reliable communications infrastructure. This project is 100% community driven and open source!

Meshtastic® 是一个让你能够利用廉价 LoRa 无线电，在缺乏现有或可靠通信基础设施的地区构建远程离线通信平台的项目。该项目完全由社区驱动且开源！

*   **远程通信：**（MartinR7 和 alleg 创下的 331 公里记录）

*   **网状网络通信无需手机**

*   **去中心化通信：**无需专用路由器

*   **加密通信**
*   **Excellent battery life**
*   **出色的电池续航**
*   **Send and receive text messages between members of the mesh**
*   **网状网络成员间可收发短信**
*   **Optional GPS based location features**
*   **可选的基于 GPS 的定位功能**
*   **And more!**
*   **以及更多功能！**

### How it works / 工作原理

Meshtastic utilizes LoRa, a long-range radio protocol, which is widely accessible in most regions without the need for additional licenses or certifications, unlike ham radio operations. These radios are designed to rebroadcast messages they receive, forming a mesh network. This setup ensures that every group member, including those at the furthest distance, can receive messages.

Meshtastic 利用 LoRa（一种远程无线电协议），该协议在大多数地区均可广泛使用，且无需像业余无线电操作那样申请额外的许可证或认证。这些无线电设备旨在转发接收到的消息，从而形成一个网状网络。这种设置确保了每一位小组成员，即使是处于最远距离的成员，也能接收到消息。

Additionally, Meshtastic radios can be paired with a single phone, allowing friends and family to send messages directly to your specific radio. It's important to note that each device is capable of supporting a connection from only one user at a time. If you are interested in a more technical overview of how Meshtastic works, visit the overview section below: Technical Overview.

此外，Meshtastic 无线电设备可以与一部手机配对，让亲朋好友能够直接向你的特定设备发送消息。需要注意的是，每台设备一次只能支持一个用户的连接。如果你对 Meshtastic 的技术原理感兴趣，请访问下方的技术概览部分：技术概览。
