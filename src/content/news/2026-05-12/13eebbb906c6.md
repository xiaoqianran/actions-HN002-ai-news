# Visual Text Compression as Measure Transport
# 视觉文本压缩：一种测度传输视角

**Abstract:** Visual text compression (VTC) promises efficient long-context processing by rendering text into an image and re-encoding it with a vision-language model, often producing $3$--$20\times$ fewer decoder tokens than subword tokenization. Yet token savings do not translate predictably into downstream utility: on some tasks the visual path matches or exceeds the text path, on others it collapses, and the compression ratio itself does not predict which regime will occur. The missing quantity is therefore not another summary of efficiency, but a principled measure of task-relevant information loss induced by visual encoding.

**摘要：** 视觉文本压缩（VTC）通过将文本渲染为图像并利用视觉语言模型进行重编码，有望实现高效的长上下文处理，其产生的解码器 Token 数量通常比子词分词（subword tokenization）少 $3$ 到 $20$ 倍。然而，Token 的节省并不能预测性地转化为下游任务的效用：在某些任务中，视觉路径的表现与文本路径相当甚至更优，但在其他任务中却会失效，且压缩比本身无法预测哪种情况会发生。因此，目前缺失的并非又一个效率指标，而是一种针对视觉编码所导致的任务相关信息损失的原则性度量。

We address this problem by formulating VTC in the language of measure transport. Treating text and visual tokens as empirical probability measures, we show that the ViT patch encoder induces a push-forward map whose transport cost decomposes into a precision cost from within-patch aggregation and a coverage cost from cross-patch fragmentation. Both terms are estimable from downstream-label-free probes.

我们通过测度传输（measure transport）的语言对 VTC 进行了建模，从而解决了这一问题。我们将文本和视觉 Token 视为经验概率测度，并证明了 ViT Patch 编码器会诱导出一个前推映射（push-forward map），其传输成本可分解为源自 Patch 内部聚合的“精度成本”和源自跨 Patch 分割的“覆盖成本”。这两项指标均可通过无需下游标签的探测（probes）进行估计。

This formulation yields two operational consequences: a downstream-label-free routing criterion that selects whether to use the visual path for a given input or benchmark instance, and a transport-informed foveation mechanism that re-encodes high-cost regions at higher resolution. Across $24$ NLP datasets at Qwen3-4B, our label-free rule matches the per-dataset oracle on $17/24$ datasets ($70.8\%$), and improves the average task score by $+3.3\%$ with $-10.3\%$ average tokens relative to a pure-LLM.

这一公式化方法带来了两个实际应用成果：一是无需下游标签的路由准则，用于选择针对特定输入或基准实例是否使用视觉路径；二是基于传输信息的中央凹（foveation）机制，能够以更高分辨率对高成本区域进行重编码。在 Qwen3-4B 模型下，针对 $24$ 个 NLP 数据集的测试显示，我们的无标签规则在 $17/24$ 个数据集（$70.8\%$）上达到了与数据集最优解（oracle）相当的效果，且在 Token 数量平均减少 $10.3\%$ 的情况下，将平均任务得分提升了 $+3.3\%$（相较于纯 LLM）。
