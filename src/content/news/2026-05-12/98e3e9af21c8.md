### Core Features
**核心功能**

AiToEarn provides four agent capabilities covering the entire monetization lifecycle for content creators: **Monetize · Publish · Engage · Create**
**AiToEarn 围绕内容创作者的完整变现链路，提供四大 Agent 能力：Monetize · Publish · Engage · Create**

*   **💰 Monetize — Content Monetization:** The core goal is to help every creator earn money. We offer three settlement models: CPS (Cost Per Sale), CPE (Cost Per Engagement), and CPM (Cost Per Mille).
    **💰 Monetize —— 内容赚钱：AiToEarn 最核心的目标：帮助每一位创作者赚钱。我们提供三种结算模式：CPS（按成交额结算）、CPE（按互动量结算）、CPM（按播放量结算）。**
*   **📢 Publish — Content Publishing Agent:** Distribute content to 10+ global platforms with one click. Includes a unified calendar for scheduling.
    **📢 Publish —— 内容发布 Agent：一键将内容分发到全球 10+ 主流平台，告别逐个平台手动发布。支持日历排期，统一规划所有平台的内容发布时间。**
*   **💬 Engage — Content Interaction Agent:** Automate interactions (likes, follows, replies) via browser plugins. Uses LLMs to generate targeted replies and identify high-conversion signals.
    **💬 Engage —— 内容互动 Agent：通过 AiToEarn 浏览器插件，在上述所有平台上实现自动化互动运营。利用 AI 智能回复生成针对性评论，并精准挖掘高转化信号。**
*   **🎨 Create — Content Creation Agent:** Reconstructs the production process. Simply state your needs, and the agent handles everything from creative ideation to final video/image output. Supports batch generation for matrix account operations.
    **🎨 Create —— 内容创作 Agent：我们用 Agent 的方式重构了内容制作流程。只需告诉 Agent 你的内容需求，它会自动完成从创意到成品的全部工作。支持批量生成，适合矩阵账号运营。**

### How to Get an API Key
**如何获取 API Key**

1.  Visit [aitoearn.cn](https://aitoearn.cn) (China) or [aitoearn.ai](https://aitoearn.ai) (Global) and log in.
    **打开 aitoearn.cn（中国用户）或 aitoearn.ai（国际用户），注册并登录。**
2.  Click "Settings" in the left menu.
    **点击左侧菜单“设置”。**
3.  Click "Create" under API Key and copy the generated key.
    **在 API Key 中点击创建，复制生成的 Key。**
