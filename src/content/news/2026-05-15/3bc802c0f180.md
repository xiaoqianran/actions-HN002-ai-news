### Introduction
### 引言

The Titanic shipwreck was a major historical incident that shaped how we view human survival during disasters. Even a century later, this tragic incident still offers valuable insights and lessons. The RMS Titanic was one of the largest and most luxurious ships of its time. It was nicknamed “The Unsinkable” by its proud makers.
泰坦尼克号沉船事件是一起重大的历史事件，它影响了我们看待灾难中人类生存的方式。即使在一个世纪后，这一悲剧事件仍然提供了宝贵的见解和教训。泰坦尼克号是当时最大、最豪华的船只之一，被其引以为傲的制造者称为“永不沉没之船”。

On April 10th, 1912, it set out on its first journey from England to New York. The Titanic took with it all classes of people, the wealthy and the poor. It was commanded by the Senior Captain Edward John Smith. During the course of its voyage, the Titanic received multiple warnings of ice on the Atlantic, which made it change its course twice. But on the 4th day of its voyage, 14th April, it collided with a huge iceberg that led to the beginning of the slow sinking of this luxurious ship.
1912年4月10日，它开始了从英国到纽约的首次航行。泰坦尼克号搭载了各个阶层的人，包括富人和穷人。它由高级船长爱德华·约翰·史密斯指挥。在航行过程中，泰坦尼克号多次收到大西洋上有冰山的警告，这使其两次改变航向。但在航行的第4天，即4月14日，它撞上了一座巨大的冰山，导致这艘豪华巨轮开始缓慢下沉。

The ship sent radio signals to other nearby ships for help, but only one of them responded. The captain ordered the passengers to be evacuated. According to the protocol, the women and children were to be evacuated first using the lifeboats available on the ship. But as we will see in our explorations, it did not really happen as such. Certain other factors also played a role in determining the survival of the passengers aboard. It seemed as if some groups of people were more likely to survive than others, and this is what we will explore in this article. The sinking of this “Unsinkable” ship caused the death of 1502 out of 2224 of its passengers and crew.
船只向附近的其他船只发送了无线电求救信号，但只有一艘船做出了回应。船长下令疏散乘客。根据协议，妇女和儿童应优先使用船上的救生艇撤离。但正如我们将在探索中看到的那样，实际情况并非完全如此。其他一些因素也在决定船上乘客的生存几率方面发挥了作用。似乎某些群体比其他人更有可能幸存，而这正是我们将在本文中探讨的内容。这艘“永不沉没”之船的沉没导致2224名乘客和船员中有1502人丧生。

### Loading the Dataset
### 加载数据集

Once you have the data URL, you can access it as a pandas dataframe. We will have to install/import pandas for this. Pandas is a powerful Python library for data analysis and manipulation. If not already installed in your IDE, install it from the terminal through pip as follows:
一旦你有了数据 URL，就可以将其作为 pandas 数据框进行访问。为此，我们需要安装/导入 pandas。Pandas 是一个强大的 Python 数据分析和处理库。如果你的 IDE 中尚未安装，请通过终端使用 pip 进行安装：

`pip install pandas`

Once the installation is complete, import the library in your Python file by aliasing it as pd:
安装完成后，在你的 Python 文件中导入该库，并将其别名为 pd：

`import pandas as pd`

Next, read the data using the Pandas read_csv function. Make sure you add the URL as follow:
接下来，使用 Pandas 的 read_csv 函数读取数据。确保添加如下 URL：

```python
url = "https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv"
df = pd.read_csv(url)
```

This will load the file as a pandas dataframe in the variable “df”. We will do the data analysis and exploration using this dataframe that has the data we need stored. Let us read the data in this dataframe using the head() function that returns the first 5 lines by default of the dataframe:
这将把文件加载为变量“df”中的 pandas 数据框。我们将使用这个存储了所需数据的数据框进行数据分析和探索。让我们使用 head() 函数读取该数据框中的数据，该函数默认返回数据框的前5行：

```python
print(df.head())
df.head()
```
