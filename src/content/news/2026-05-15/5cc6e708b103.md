# OpenAI says Codex is coming to your phone
# OpenAI 宣布 Codex 即将登陆手机端

OpenAI says Codex is coming to your phone. Codex is going mobile. The coding tool — which OpenAI launched approximately a year ago — has now been integrated into the ChatGPT app, allowing users to monitor and manage their development workflows remotely.
OpenAI 宣布 Codex 即将登陆手机端。Codex 正在走向移动化。这款由 OpenAI 大约一年前推出的编程工具，现已集成到 ChatGPT 应用程序中，允许用户远程监控和管理他们的开发工作流程。

The new function allows users to see their Codex live environments in any devices where it is running. The company announced the changes Thursday; the update, which is currently in preview, is now available to all plans on iOS and Android.
这项新功能允许用户在任何运行 Codex 的设备上查看其实时环境。该公司于周四宣布了这一变动；该更新目前处于预览阶段，现已向 iOS 和 Android 平台上的所有订阅计划用户开放。

“This is more than the ability to remotely control a single task or dispatch new tasks to your computer,” OpenAI said in a statement. “From your phone, you can work across all of your threads, review outputs, approve commands, change models, or start something new.”
“这不仅仅是远程控制单个任务或向电脑发送新任务的能力，”OpenAI 在一份声明中表示。“通过手机，你可以处理所有的线程、审查输出结果、批准指令、切换模型或开启全新的工作。”

Last month, OpenAI also gave Codex the ability to run in the background in desktop environments — empowering the tool to take care of various tasks autonomously. Earlier this month, the company also introduced a Chrome extension that allows the agent to work in live browser sessions.
上个月，OpenAI 还赋予了 Codex 在桌面环境中后台运行的能力，使该工具能够自主处理各种任务。本月早些时候，该公司还推出了一款 Chrome 扩展程序，允许该智能体在实时浏览器会话中工作。

In February, Anthropic released a similar feature — Remote Control — which allows users to remotely monitor Claude Code’s work from afar. The flurry of feature releases from both OpenAI and Anthropic speaks to the tense competition between the two over whose agentic coding tool will become the most widely used.
今年 2 月，Anthropic 发布了类似的功能——“远程控制”（Remote Control），允许用户从远处监控 Claude Code 的工作。OpenAI 和 Anthropic 近期密集发布的功能更新，凸显了双方在争夺“谁的智能编程工具将成为最广泛使用工具”这一目标上的激烈竞争。

Over the past year, Anthropic’s Claude Code has gained in popularity among businesses and tech professionals alike, although both tools continue to be widely used.
在过去的一年中，Anthropic 的 Claude Code 在企业和技术专业人士中广受欢迎，尽管这两款工具目前都在被广泛使用。
