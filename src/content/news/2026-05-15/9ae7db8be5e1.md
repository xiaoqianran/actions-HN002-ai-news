Welcome to Classic 7! Classic 7 is a Windows 10 (IoT Enterprise LTSC 2021) modification made to look 1:1 to Windows 7. It has all of the goodies that Windows 7 had along with some extras included!
欢迎来到 Classic 7！Classic 7 是基于 Windows 10 (IoT Enterprise LTSC 2021) 的修改版，旨在 1:1 复刻 Windows 7 的外观。它不仅保留了 Windows 7 的所有经典元素，还额外加入了一些新功能！

Classic 7 features a 1:1 OOBE recreation, meaning it'll feel just like your PC simplified once more.
Classic 7 实现了 1:1 的开箱体验（OOBE）重现，这意味着你的电脑将再次回归那种简洁的操作感。

What features does Classic 7 contain? Classic 7 comes with most of the features that Windows 7 had, along with some exceptions that may or may not happen to be included in the near future.
Classic 7 包含哪些功能？它内置了 Windows 7 的大部分功能，同时也存在一些例外，这些功能未来可能会（或可能不会）被加入。

Some features included from Windows 7 are: Aero Glass! Support for .themepack files from Windows 7! All of the beloved gadgets you remember! Windows Media Center is fully intact and functional! And of course, the beautiful theme elements from Windows 7!
其中包含的部分 Windows 7 功能包括：Aero Glass 特效！支持 Windows 7 的 .themepack 主题包！你记忆中所有深受喜爱的桌面小工具！完整且功能正常的 Windows Media Center！当然，还有 Windows 7 那精美的主题元素！
