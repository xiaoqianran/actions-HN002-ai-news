# CAWI: Copula-Aligned Weight Initialization for Randomized Neural Networks
# CAWI：用于随机神经网络的 Copula 对齐权重初始化

**Abstract:** Randomized neural networks (RdNNs) enable efficient, backpropagation-free training by freezing randomly initialized input-to-hidden weights, which permits a closed-form solution for the output layer. However, conventional random initialization is blind to inter-feature dependence, ignoring correlations, asymmetries, and tail dependence in the data, which degrades conditioning and predictive performance. To the best of our knowledge, this limitation remains unaddressed in the RdNN literature.

**摘要：** 随机神经网络（RdNNs）通过冻结随机初始化的输入到隐藏层权重，实现了高效且无需反向传播的训练，从而允许对输出层进行闭式求解。然而，传统的随机初始化方法对特征间的依赖关系视而不见，忽略了数据中的相关性、非对称性和尾部依赖，这降低了模型的条件数和预测性能。据我们所知，这一局限性在现有的 RdNN 文献中尚未得到解决。

To close this gap, we propose CAWI (Copula-Aligned Weight Initialization), a framework that draws input-to-hidden weights from a data-fitted copula that matches empirical dependence, ensuring the frozen projections respect inter-feature dependence without sacrificing the closed-form solution. CAWI (i) maps each feature to the unit interval using empirical CDFs, (ii) fits a multivariate copula that captures rank-based dependence among features, and (iii) samples each weight column w_j from the fitted copula and applies a fixed inverse marginal transform to set scale. The objective, solver, and "freeze-once" paradigm remain unchanged; only the sampling law for W becomes dependence-aware.

为了弥补这一空白，我们提出了 CAWI（Copula 对齐权重初始化），这是一个从拟合数据的 Copula 中抽取输入到隐藏层权重的框架。该框架能够匹配经验依赖关系，确保冻结的投影在不牺牲闭式求解的前提下，尊重特征间的依赖性。CAWI 的具体步骤为：(i) 使用经验累积分布函数（CDF）将每个特征映射到单位区间；(ii) 拟合一个能够捕捉特征间基于秩的依赖关系的多元 Copula；(iii) 从拟合的 Copula 中采样每个权重列 w_j，并应用固定的逆边际变换来设定尺度。其目标函数、求解器和“一次冻结”范式保持不变；仅权重矩阵 W 的采样规律变得具有依赖感知能力。

For dependence modeling, we consider two copula families: elliptical (Gaussian, t) and Archimedean (Clayton, Frank, Gumbel). This enables CAWI to handle diverse dependence, including tail dependence. We evaluate CAWI across 83 diverse classification benchmarks (binary and multiclass) and two biomedical datasets, BreaKHis and the Schizophrenia dataset, using standard shallow and deep RdNN architectures. CAWI consistently delivers significant improvements in predictive performance over conventional random initialization. Code is available at: this https URL.

在依赖建模方面，我们考虑了两个 Copula 族：椭圆型（高斯、t 分布）和阿基米德型（Clayton、Frank、Gumbel）。这使得 CAWI 能够处理多种依赖关系，包括尾部依赖。我们使用标准的浅层和深层 RdNN 架构，在 83 个不同的分类基准测试（二分类和多分类）以及两个生物医学数据集（BreaKHis 和精神分裂症数据集）上对 CAWI 进行了评估。结果表明，与传统的随机初始化相比，CAWI 在预测性能上始终有显著提升。代码地址：[点击此处](https://github.com/mushir-akhtar/CAWI)。
