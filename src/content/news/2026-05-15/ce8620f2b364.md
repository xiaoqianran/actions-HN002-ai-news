### Notion Developer Platform

**Notion Developer Platform**
Notion 开发者平台

**Notion is an all-in-one workspace that combines note-taking, project management, and task organization. It allows users to create customized databases, documents, and calendars to streamline their personal and professional workflows.**
Notion 是一款集笔记、项目管理和任务组织于一体的一站式工作空间。它允许用户创建自定义数据库、文档和日历，从而简化个人及专业工作流程。

**Notion Developer Platform lets teams build on Notion with CLI, Workers, database syncs, agent tools, webhook triggers, MCP, and External Agents APIs, so data, workflows, and agents can operate inside the same shared workspace.**
Notion 开发者平台允许团队利用 CLI、Workers、数据库同步、智能体工具、Webhook 触发器、MCP 以及外部智能体 API 在 Notion 之上进行构建，使数据、工作流和智能体能够在同一个共享工作空间内协同运行。
