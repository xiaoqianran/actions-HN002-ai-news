# HKUDS / CLI-Anything

**CLI-Anything: Making ALL Software Agent-Native**
**CLI-Anything：让所有软件都具备“智能体原生”能力**

Today's Software Serves Humans👨‍💻. Tomorrow's Users will be Agents🤖.
今天的软件服务于人类👨‍💻。明天的用户将是智能体🤖。

CLI-Hub: `pip install cli-anything-hub` then `cli-hub install <name>` — browse, install, and manage all community-built CLIs. Want to add your own? Open a PR — the hub updates instantly.
CLI-Hub：通过 `pip install cli-anything-hub` 安装，随后使用 `cli-hub install <name>` —— 即可浏览、安装并管理所有社区构建的 CLI 工具。想要添加你自己的工具？提交一个 PR 即可 —— Hub 会即时更新。

🎬 查看演示：观看 AI 智能体如何使用生成的 CLI 工具，结合预览、实时预览和轨迹循环来产出实际成果 —— 包括 CAD 构建、3D 场景、图表、游戏画面、字幕等。
