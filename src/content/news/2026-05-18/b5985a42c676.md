# New design for the FreeBSD website
# FreeBSD 网站全新设计

A Powerful, Open Source Operating System. Download FreeBSD 15.0 (amd64, aarch64, other). Announcement, notes.
一个强大的开源操作系统。下载 FreeBSD 15.0（amd64、aarch64 及其他架构）。查看公告与发行说明。

Native ZFS: OpenZFS combines the roles of a filesystem and volume manager into a single, reliable storage platform. In FreeBSD, it is fully integrated into the operating system.
原生 ZFS：OpenZFS 将文件系统和卷管理器的角色结合为一个单一、可靠的存储平台。在 FreeBSD 中，它已与操作系统完全集成。

Virtualization: bhyve is FreeBSD’s modern hypervisor, with support for virtio, PCI passthrough, and fast NVMe emulation.
虚拟化：bhyve 是 FreeBSD 的现代虚拟机管理程序，支持 virtio、PCI 直通和快速 NVMe 模拟。

Jails: FreeBSD jails provide isolation for services and applications. Each jail can run with its own filesystem, networking, and configuration.
Jails：FreeBSD Jails 为服务和应用程序提供隔离环境。每个 Jail 都可以拥有独立的文件系统、网络和配置。

Networking: FreeBSD is widely recognized for its network performance and stability. It powers infrastructure that handles demanding workloads around the world.
网络：FreeBSD 以其网络性能和稳定性而广受认可。它为全球处理高负载任务的基础设施提供动力。

Documentation: FreeBSD includes extensive, well-maintained documentation. From the Handbook to manual pages, documentation is treated as a core part of the system.
文档：FreeBSD 包含详尽且维护良好的文档。从《手册》到说明文档（man pages），文档被视为系统的核心组成部分。

Community: FreeBSD has a vibrant and collaborative community, from individual contributors to large corporate organisations. Groups are active across various channels, from real-time communications to mailing lists and newsgroups.
社区：FreeBSD 拥有一个充满活力且协作性强的社区，成员涵盖个人贡献者到大型企业组织。各小组活跃在各种渠道中，从实时通讯到邮件列表和新闻组。
