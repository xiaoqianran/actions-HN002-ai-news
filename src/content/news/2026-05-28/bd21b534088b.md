# Frequency-Guided Fusion For RGB-Thermal Semantic Segmentation
# 基于频率引导融合的RGB-热成像语义分割

**Abstract:** Semantic segmentation in complex environments such as urban driving scenes remains challenging under adverse lighting conditions, where RGB images alone provide insufficient information. RGB-Thermal fusion leverages the complementary strengths of visible and infrared imagery to improve scene understanding; however, effectively integrating these heterogeneous modalities at varying levels of feature abstraction remains an open problem.

**摘要：** 在城市驾驶场景等复杂环境中，语义分割在恶劣光照条件下仍然极具挑战性，仅靠RGB图像往往无法提供足够的信息。RGB-热成像融合技术利用可见光和红外图像的互补优势来提升场景理解能力；然而，如何在不同特征抽象层级上有效地整合这些异构模态，仍然是一个亟待解决的问题。

In this paper, we propose a multi-modal fusion architecture built upon dual ConvNeXt V2 backbones that employs stage-wise, modality-adaptive fusion strategies. For early-stage features, we introduce a Frequency-Based Fusion Module that decomposes infrared features into low- and high-frequency components via Gaussian filtering, applies dual-branch spatial attention to selectively emphasize thermal patterns and fine-grained boundaries, and integrates them with RGB features through a confidence-gated residual mechanism.

在本文中，我们提出了一种基于双ConvNeXt V2主干网络的多模态融合架构，该架构采用了分阶段、模态自适应的融合策略。针对早期特征，我们引入了一个基于频率的融合模块（Frequency-Based Fusion Module），通过高斯滤波将红外特征分解为低频和高频分量，应用双分支空间注意力机制来选择性地强调热成像模式和细粒度边界，并通过置信度门控残差机制将其与RGB特征进行整合。

For late-stage features, we design a semantic fusion module with cross-modal attention and multi-scale depthwise convolutions to capture semantic correspondences across modalities. The fused features are decoded via a PANet-style bidirectional decoder with deep supervision.

针对后期特征，我们设计了一个语义融合模块，利用跨模态注意力和多尺度深度卷积来捕捉模态间的语义对应关系。融合后的特征通过带有深度监督的PANet风格双向解码器进行解码。

Experiments on MFNet and PST900 demonstrate that our lightest variant achieves 61.73% and 86.24% mIoU, respectively, with only 35.43M parameters, outperforming recent methods while using substantially fewer parameters and lower computational cost. Code is available at this https URL.

在MFNet和PST900数据集上的实验表明，我们最轻量级的变体分别达到了61.73%和86.24%的mIoU，且参数量仅为35.43M。该方法在参数量更少、计算成本更低的情况下，性能优于现有的主流方法。代码已在链接中提供。
