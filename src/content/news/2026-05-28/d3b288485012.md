# SimCity 3k in 4k (2025)

SimCity 3000 is the best SimCity. It's built upon a souped-up version of the Sc2K engine. It features isometric art, crafted pixel by pixel. It strikes the right balance of features and complexity; it feels like it could be mastered but without coddling you. It also has a high resolution Jazzy/new age OST.
《模拟城市 3000》是该系列中最棒的作品。它基于《模拟城市 2000》引擎的强化版本构建，拥有逐像素精心绘制的等距视角美术风格。它在功能与复杂性之间取得了完美的平衡；既让人觉得可以精通，又不会显得过于简单。此外，它还拥有高音质的爵士/新世纪风格原声带。

Running it on modern systems takes a few tweaks but is certainly doable. By "modern" I mean Windows 10 LTSC 2021, Ryzen 5 3600, Radeon Rx 7600, 48 GB of RAM, 4K monitor.
在现代系统上运行它需要一些调整，但绝对可行。我所说的“现代”配置是指：Windows 10 LTSC 2021、Ryzen 5 3600、Radeon Rx 7600、48 GB 内存以及 4K 显示器。

I first installed SC3k from its CD ROM. I've had this Scholastic Edition CD all my life, I actually purchased it at a school book fair in a primary school building that no longer exists. But the disk still does, and I will not pay for the same game twice. The installation completed without trouble on 64-bit Windows 10.
我最初是从 CD-ROM 安装 SC3k 的。这张“Scholastic 版”光盘我保留了一辈子，它是我在一家早已不复存在的小学书展上买的。但光盘还在，我绝不会为同一款游戏付两次钱。在 64 位 Windows 10 上，安装过程非常顺利。

The game launched but it wasn't very playable: There was no widescreen support; the resolutions supported were relics like 800x600 and 1024x768; the game was laggy when scrolling; the game accelerated when scrolling; there is a big slowdown after loading a city; the music doesn't play without using the CD, and some of it was missing! Some tiles had missing graphics for a few seconds before their contents were drawn. Not to worry. There are remedies for each of these.
游戏虽然启动了，但体验并不好：没有宽屏支持；支持的分辨率都是 800x600 和 1024x768 这样的老古董；滚动时画面卡顿；滚动时速度又过快；加载城市后会有明显的延迟；不放光盘就没音乐，而且部分音乐缺失！有些地块在内容加载出来前，会有几秒钟的图形丢失。不过别担心，这些问题都有解决办法。

By the end of this article, you will have a SimCity 3000 game that looks just like this: Behold, SimCity 3000 in 4K! Risky biscuits: building 1WTC and 2WTC but no hospitals or fire stations. I like to tease fate.
读完本文，你将拥有一款看起来像这样的《模拟城市 3000》：看，这就是 4K 分辨率下的《模拟城市 3000》！冒险尝试：只建世贸中心 1 号和 2 号楼，却不建医院或消防站。我就是喜欢挑战命运。

### 1. GOG Patched EXE
The patched EXE will enable widescreen support. It also acts as a No-CD patch. GOG.com contributed this patch and have shared it for a public download. Download this and replace the original EXE with it. Now you can start the game and choose a larger resolution. However, the mouse acceleration will now be too much. Also, the game will appear horribly askew if you've chosen a resolution that has a factor of 8, which is very likely because that's most of them. Not to worry. We can fix that too.
### 1. GOG 补丁版 EXE
这个补丁后的 EXE 文件将启用宽屏支持，同时也是一个免光盘补丁。GOG.com 贡献了这个补丁并公开供大家下载。下载并替换原有的 EXE 文件后，你就可以启动游戏并选择更高的分辨率了。不过，此时鼠标加速会变得非常严重。此外，如果你选择的分辨率是 8 的倍数（这很常见，大部分分辨率都是），游戏画面会变得极其扭曲。别担心，我们也能修复这个问题。

### 2. Tweak the mouse acceleration
Edit the SC3U.ini file. Modify or set this variable:
`[Navigation]`
`ScrollMarginFactor = 0.005787`
### 2. 调整鼠标加速
编辑 SC3U.ini 文件，修改或设置以下变量：
`[Navigation]`
`ScrollMarginFactor = 0.005787`

### 3. Install the D3D Wrapper Alternative
This will fix the resolution weirdness, meaning you can play the game at full 4k. My monitor is 3840x2160 native, and this will make the game support it. It will also fix any other graphical acceleration problems on modern cards and DirectX 11+. Extract these files to your SC3K Apps directory. That's it. Almost...
### 3. 安装 D3D Wrapper 替代方案
这将修复分辨率异常的问题，让你能以完整的 4K 分辨率进行游戏。我的显示器原生分辨率是 3840x2160，安装后游戏就能完美支持。它还能解决现代显卡和 DirectX 11+ 环境下的其他图形加速问题。将这些文件解压到你的 SC3K Apps 目录下即可。搞定，差不多了……

**Configure the D3dWrapper**
By default the D3d Wrapper will run in borderless, not fullscreen. That means your Taskbar will push the game off the screen! We will switch it to true fullscreen. Inside the SC3K Apps directory there is a file dxwrapper.ini. Make the following changes:
`[d3d9]`
`EnableWindowMode = 0`
`...`
`[FullScreen]`
`FullScreen = 1`
I also did this: `EnableVsync = 1`
**配置 D3dWrapper**
默认情况下，D3d Wrapper 会以无边框模式运行，而不是全屏。这意味着任务栏会把游戏窗口挤出屏幕！我们要把它切换为真正的全屏模式。在 SC3K Apps 目录下找到 dxwrapper.ini 文件，进行以下修改：
`[d3d9]`
`EnableWindowMode = 0`
`...`
`[FullScreen]`
`FullScreen = 1`
我还设置了：`EnableVsync = 1`

### 4. Install the 4GB patch
When scrolling the map, you'll see tiles appear as a small pale cube before their contents are loaded into view. Installing the 4GB patch will give the game more RAM to play with, and should help. Download NTCore and run this on the SC3K.exe file. It will allow the game to use more RAM on 64-bit systems.
### 4. 安装 4GB 补丁
滚动地图时，你会看到地块在内容加载出来前显示为一个苍白的小方块。安装 4GB 补丁可以让游戏使用更多内存，应该会有所改善。下载 NTCore 并对 SC3K.exe 文件运行它，这将允许游戏在 64 位系统上使用更多内存。

### 5. Disable the auto updater
The slowdown when first opening the game is caused by the auto-updater trying to contact a long-dead update server. You can disable it with this patched .ini file. Rename it to `UpdateSettings.ini` and stick it in the `Apps/Updater` directory.
### 5. 禁用自动更新程序
游戏首次打开时的卡顿是因为自动更新程序试图连接早已失效的更新服务器。你可以使用这个补丁后的 .ini 文件来禁用它。将其重命名为 `UpdateSettings.ini` 并放入 `Apps/Updater` 目录中即可。

### 6. Missing Music on SimCity 3000 Unlimited
Some of the music from the original release is missing from an .ini file, even though it is present in Unlimited. Fix that with the following patch. Finally, if you installed SC3K from CD like me, the music is not copied to your computer. Fix that by navigating to the CD directory: `APPS/RES/SOUND/MUSIC` then copy the .xa files to the same directory on your HDD. Done!
### 6. 修复《模拟城市 3000 典藏版》缺失的音乐
原版游戏中的部分音乐在 .ini 文件中丢失了，尽管它们实际上存在于典藏版中。使用以下补丁修复它。最后，如果你像我一样是从光盘安装的，音乐文件并没有被复制到电脑上。解决方法是进入光盘目录：`APPS/RES/SOUND/MUSIC`，然后将 .xa 文件复制到硬盘上的对应目录中。大功告成！
