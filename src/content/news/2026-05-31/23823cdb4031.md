# NixOS 26.05 发布

Hey everyone, we are yayayayaka and jopejoe1, the release managers of the newest release of NixOS. We are very proud to announce the public availability of NixOS 26.05 “Yarara”.
大家好，我们是 yayayayaka 和 jopejoe1，是 NixOS 最新版本的发布经理。我们非常自豪地宣布 NixOS 26.05 “Yarara” 正式发布。

NixOS is a Linux distribution. Its underlying package repository Nixpkgs can also be used on other Linux systems and macOS with the Nix package manager. This release will receive bugfixes and security updates for seven months (up until 2026-12-31). The old release 25.11 “Xantusia” is now officially deprecated and will reach its end-of-life and stop receiving security updates after 2026-06-30.
NixOS 是一个 Linux 发行版。其底层的软件包仓库 Nixpkgs 也可以通过 Nix 包管理器在其他 Linux 系统和 macOS 上使用。此版本将获得为期七个月的错误修复和安全更新（直至 2026 年 12 月 31 日）。旧版本 25.11 “Xantusia” 现已正式弃用，并将于 2026 年 6 月 30 日后停止接收安全更新，正式结束生命周期。

NixOS Release Notes Highlights, New Modules, Backward Incompatibilities, Other Notable Changes. Nixpkgs Release Notes Highlights, Backward Incompatibilities, Other Notable Changes, Nixpkgs Library, Upgrade Instructions, ISO Image Downloads.
NixOS 发行说明：亮点、新模块、向后不兼容性、其他重要变更。Nixpkgs 发行说明：亮点、向后不兼容性、其他重要变更、Nixpkgs 库、升级指南、ISO 镜像下载。

The 26.05 release was made possible due to the efforts of 2842 contributors, who authored 59703 commits since the previous release.
26.05 版本的发布离不开 2842 位贡献者的努力，他们在上一个版本发布后共提交了 59703 次代码。

Highlights: Our vast and routinely maintained set of packages has also been updated. This release of Nixpkgs Added 20442 new packages, Updated 20641 existing packages, Removed 17532 outdated packages, in an effort to keep the package set maintainable and secure.
亮点：我们庞大且定期维护的软件包集也得到了更新。此版本的 Nixpkgs 新增了 20442 个软件包，更新了 20641 个现有软件包，并移除了 17532 个过时软件包，旨在保持软件包集的可维护性和安全性。

In addition to packages from Nixpkgs, the NixOS Linux distribution also features composable configuration modules and integration tests for distributed systems. This release of NixOS Added 85 new modules and 1547 configuration options, Removed 25 outdated modules and 355 configuration options.
除了来自 Nixpkgs 的软件包外，NixOS Linux 发行版还具有可组合的配置模块和分布式系统集成测试功能。此版本的 NixOS 新增了 85 个模块和 1547 个配置选项，移除了 25 个过时模块和 355 个配置选项。

systemd stage 1: Stage 1 (a.k.a. initrd) is now based on systemd by default, and the old scripted implementation is deprecated and scheduled for removal in 26.11.
systemd 第一阶段：第一阶段（即 initrd）现在默认基于 systemd，旧的脚本实现已被弃用，并计划在 26.11 版本中移除。

Deprecation of x86_64-darwin: This will be the last release of Nixpkgs to support x86_64-darwin. Platform support will be maintained and binaries built until Nixpkgs 26.05 goes out of support at the end of 2026. For 26.11, due to Apple’s deprecation of the platform and limited build infrastructure and developer time, we will no longer build packages for x86_64-darwin or support building them from source.
弃用 x86_64-darwin：这将是 Nixpkgs 支持 x86_64-darwin 的最后一个版本。平台支持和二进制构建将持续到 2026 年底 Nixpkgs 26.05 停止支持为止。对于 26.11 版本，由于苹果公司已弃用该平台，且构建基础设施和开发者时间有限，我们将不再为 x86_64-darwin 构建软件包，也不再支持从源码构建。

GNOME 50: GNOME has been updated to version 50 “Tokyo”, introducing accessibility enhancements, display handling improvements and more. Refer to the release notes for more details.
GNOME 50：GNOME 已更新至 50 版本 “Tokyo”，引入了辅助功能增强、显示处理改进等功能。详情请参阅发行说明。

C compilers: GCC has been updated to version 15. LLVM remains at version 21.
C 编译器：GCC 已更新至 15 版本。LLVM 保持在 21 版本。

We also want to thank all contributors who made this release possible!
特别感谢：我们要特别感谢 Bryan Honof 和 raf 对发行说明的编辑工作；感谢 Yohann Boniface 设计的发布 Logo；感谢 NixOS 基础设施团队尽职尽责地维护我们的构建基础设施；感谢 Nixpkgs 暂存团队支持我们的暂存周期并耐心修复了许多构建错误。我们还要感谢所有促成此次发布的贡献者！

Seeing all the contributors working in their area of the project to improve it has been an exciting experience. We would like to thank everyone in the community for that.
