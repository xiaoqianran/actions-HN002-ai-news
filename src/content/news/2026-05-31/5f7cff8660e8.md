# Linear Diffs

Linear is the product development system for teams and agents.
Linear 是专为团队和智能体（Agents）打造的产品开发系统。

Linear is a new species of product tool. Purpose-built for product teams with agent workflows at its core, it sets a new standard for planning and building products. Linear is the tool of choice for 20,000+ companies including OpenAI, Ramp, and Vercel.
Linear 是一种全新的产品工具。它以智能体工作流为核心，专为产品团队打造，为产品规划与构建设立了新标准。Linear 是包括 OpenAI、Ramp 和 Vercel 在内的超过 20,000 家公司的首选工具。

Introducing Linear Diffs to make reviewing code a fast and fluid experience native to Linear. You can now review diffs from any issue with a PR, iterate on further changes with agents, and ship code from Linear.
Linear 现已推出 Linear Diffs，旨在让代码审查成为一种在 Linear 内原生、快速且流畅的体验。现在，你可以直接在关联了 PR 的议题（Issue）中审查代码差异（diffs），利用智能体进行后续迭代，并直接从 Linear 发布代码。

All reviews in Linear sync back to GitHub, so the current state of review work is always clear. Linear Diffs is available now on all plans.
