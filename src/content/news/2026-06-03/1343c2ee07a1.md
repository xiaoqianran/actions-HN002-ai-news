# Improved Belief-Attention in Vision Task
# 视觉任务中改进的信念注意力机制 (Belief-Attention)

**Abstract:** Recently, Belief-Attention \cite{Guoqiang25BeliefAttention} has been proposed by first performing an orthogonal projection of the softmax-based weighted summation of $V$ vectors with respect to the original $V$ vectors and then taking the perpendicular component as the residual signal in Transformer for performance improvement.

**摘要：** 最近，研究人员提出了信念注意力机制（Belief-Attention），其方法是首先对基于 Softmax 的 $V$ 向量加权求和结果，相对于原始 $V$ 向量进行正交投影，然后将垂直分量作为 Transformer 中的残差信号，从而提升性能。

In this paper, we first conduct an ablation study showing the projected component also carries information about the token correlation, which should not be ignored. We then propose to extend Belief-Attention by making use of both the perpendicular and projected components. In particular, the projected component goes through certain activation function and then a linear mapping before merging with the considered token.

在本文中，我们首先进行了消融研究，结果表明投影分量同样携带了关于 Token 相关性的信息，这一点不应被忽视。随后，我们提出通过同时利用垂直分量和投影分量来扩展信念注意力机制。具体而言，投影分量在与目标 Token 合并之前，会经过特定的激活函数和线性映射处理。

Conceptually speaking, the neural block for the projected component can be viewed as a two-layer feedforward network (FFN) within the new attention block. It is also noted that standard attention captures the token correlation via the inner-product matrix $QK^T$. We propose to introduce an additional inner-product matrix $ZZ^T$ to $QK^T$ to capture richer token correlation. We refer to the new module as Belief2-Attention.

从概念上讲，用于处理投影分量的神经模块可以被视为新注意力模块中的一个两层前馈网络（FFN）。此外，值得注意的是，标准注意力机制通过内积矩阵 $QK^T$ 来捕捉 Token 间的相关性。我们提出在 $QK^T$ 中引入一个额外的内积矩阵 $ZZ^T$，以捕捉更丰富的 Token 相关性。我们将这一新模块称为 Belief2-Attention。

It can be easily shown that Belief2-Attention is more expressive than standard Attention. We then verify the effectiveness of Belief2-Attention for vision tasks of image classification and segmentation.

研究表明，Belief2-Attention 比标准注意力机制具有更强的表达能力。最后，我们验证了 Belief2-Attention 在图像分类和分割等视觉任务中的有效性。
