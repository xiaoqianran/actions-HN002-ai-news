# Novel Aspects of IEEE SA P3109 Arithmetic Formats for Machine Learning
# IEEE SA P3109 机器学习算术格式的新颖方面

**Abstract:** The IEEE P3109 draft standard defines a parameterized family of binary floating-point formats and associated operations, with a focus on facilitating machine learning. These formats allow efficient and consistent representation of values in a small number of bits.
**摘要：** IEEE P3109 标准草案定义了一系列参数化的二进制浮点格式及相关运算，旨在促进机器学习的发展。这些格式允许以较少的位数对数值进行高效且一致的表示。

The defined formats are parameterized over width and precision in bits, signedness, and the presence of infinities. Operations are defined by decoding floating-point values to the set of closed extended reals: the reals augmented with positive and negative infinity and NaN (Not a Number).
所定义的格式在位宽、精度、符号位以及是否包含无穷大等方面均可进行参数化配置。运算的定义方式是将浮点值解码为闭合扩展实数集：即在实数基础上增加了正无穷、负无穷和 NaN（非数值）。

Explicit treatment of NaN and infinite operands ensures that only real arithmetic is invoked in operation definitions. Extensive rounding and saturation modes are defined; stochastic rounding is included.
对 NaN 和无穷大操作数的显式处理，确保了运算定义中仅调用实数算术。标准中定义了广泛的舍入和饱和模式，并包含了随机舍入。

Operations are exception-free, accelerating throughput, with exceptional situations communicated through return values, e.g., NaN. Operations on blocks of values sharing a common scale factor are defined in terms of the underlying operations in a uniform manner.
运算过程不产生异常，从而提高了吞吐量；异常情况通过返回值（例如 NaN）进行传递。对于共享公共比例因子的数值块，其运算方式以统一的方式基于底层运算进行定义。

System vendors may describe approximate implementations via a novel scale-invariant measure, akin to units in the last place, called kappa-approximation. Standard function definitions and various other properties are mechanically verified and generated using formal specifications.
系统供应商可以通过一种名为“kappa 近似”（kappa-approximation）的新型尺度不变度量来描述近似实现，这类似于最后一位单位（ULP）。标准函数定义及其他各种属性均通过形式化规范进行机械验证和生成。
