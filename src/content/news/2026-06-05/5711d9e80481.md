# Self-Distilled Policy Gradient

**Abstract:** On-policy self-distillation, where a language model conditions on privileged context to supervise its own generations, is a promising source of dense supervision for sparse-reward reinforcement learning.

**摘要：** 在策略自蒸馏（On-policy self-distillation）是一种为稀疏奖励强化学习提供密集监督的有前景的方法，其核心在于让语言模型基于特权上下文（privileged context）来监督自身的生成过程。

Actually, it can be instantiated as an auxiliary full-vocabulary student-to-teacher reverse Kullback-Leibler divergence loss.

实际上，它可以被实例化为一种辅助性的全词表学生到教师的反向 Kullback-Leibler 散度损失（reverse KL divergence loss）。

We therefore propose SDPG, a self-distilled policy-gradient framework that combines group-relative verifier advantages with normalized standard deviation, exact full-vocabulary on-policy self-distillation, as well as reference-policy KL regularization.

因此，我们提出了 SDPG，这是一个自蒸馏策略梯度框架。该框架结合了组相对验证器优势（group-relative verifier advantages）、归一化标准差、精确的全词表在策略自蒸馏，以及参考策略 KL 正则化。

Empirically, SDPG improves stability and performance over RLVR and self-distillation baselines. The code is available at this https URL.

实验结果表明，与 RLVR 和自蒸馏基线相比，SDPG 提高了训练的稳定性和性能。代码已在链接中提供。
