# Early Detection of Alzheimer's Disease Using Explainable Machine Learning on Clinical Biomarkers: A Multi-Class Classification Study Using the Alzheimer's Disease Neuroimaging Initiative (ADNI) Dataset

**利用可解释机器学习与临床生物标志物早期检测阿尔茨海默病：基于阿尔茨海默病神经影像学倡议 (ADNI) 数据集的多分类研究**

---

**Abstract:**
**Background:** Alzheimer's disease (AD) affects over 55 million people worldwide. Accurate, interpretable detection of normal cognition (NC), mild cognitive impairment (MCI), and AD from routine clinical assessments remains a critical unmet need.

**摘要：**
**背景：** 阿尔茨海默病 (AD) 影响着全球超过 5500 万人。如何通过常规临床评估准确且可解释地识别正常认知 (NC)、轻度认知障碍 (MCI) 和阿尔茨海默病，仍然是一个亟待解决的关键需求。

**Methods:** An XGBoost classifier was developed for three-class detection using eight clinical features from the Alzheimer's Disease Neuroimaging Initiative (ADNI): MMSE, CDR Global, CDR Sum of Boxes (CDR-SB), MoCA, FAQ, age, sex, and education. Hyperparameters were optimised using Optuna (50 trials); class imbalance was addressed with SMOTE. Performance was evaluated by macro AUC-ROC with 1,000-iteration bootstrap 95% confidence intervals, macro F1, balanced accuracy, and Cohen's kappa. SHAP values provided feature-level explainability.

**方法：** 本研究开发了一种 XGBoost 分类器，利用来自阿尔茨海默病神经影像学倡议 (ADNI) 的八项临床特征进行三分类检测，包括：简易精神状态检查 (MMSE)、临床痴呆评定量表 (CDR) 全局评分、CDR 总分 (CDR-SB)、蒙特利尔认知评估 (MoCA)、功能活动问卷 (FAQ)、年龄、性别和教育程度。研究使用 Optuna 优化了超参数（50 次试验），并采用 SMOTE 技术处理类别不平衡问题。性能评估指标包括宏观 AUC-ROC（通过 1000 次迭代的自助法计算 95% 置信区间）、宏观 F1 分数、平衡准确率和 Cohen's kappa 系数。SHAP 值则提供了特征层面的可解释性。

**Results:** The dataset comprised 1,641 baseline subjects (608 NC, 767 MCI, 266 AD). On five-fold cross-validation, mean macro AUC was 0.983 (SD 0.007), accuracy 0.944 (SD 0.006), and macro F1 0.929 (SD 0.008). On the held-out test set (n = 247), macro AUC was 0.982 (95% CI: 0.965--0.995), accuracy 0.943, balanced accuracy 0.932, macro F1 0.927, and Cohen's kappa 0.909. SHAP analysis identified CDR Global as the dominant predictor for NC and MCI, while CDR-SB and MMSE together drove AD classification.

**结果：** 数据集包含 1,641 名基线受试者（608 名 NC，767 名 MCI，266 名 AD）。在五折交叉验证中，平均宏观 AUC 为 0.983（标准差 0.007），准确率为 0.944（标准差 0.006），宏观 F1 分数为 0.929（标准差 0.008）。在留出测试集 (n = 247) 上，宏观 AUC 为 0.982（95% 置信区间：0.965--0.995），准确率为 0.943，平衡准确率为 0.932，宏观 F1 分数为 0.927，Cohen's kappa 系数为 0.909。SHAP 分析显示，CDR 全局评分是识别 NC 和 MCI 的主要预测因子，而 CDR-SB 和 MMSE 共同驱动了 AD 的分类。

**Conclusion:** An explainable machine learning model trained on routine clinical assessments achieves near-perfect three-class Alzheimer's detection. SHAP analysis reveals clinically plausible, class-specific feature importance patterns supporting clinical validity. Future work will extend this framework with speech biomarkers for multimodal detection.

**结论：** 基于常规临床评估训练的可解释机器学习模型实现了近乎完美的三分类阿尔茨海默病检测。SHAP 分析揭示了具有临床合理性的、针对特定类别的特征重要性模式，从而支持了其临床有效性。未来的工作将通过引入语音生物标志物来扩展该框架，以实现多模态检测。
