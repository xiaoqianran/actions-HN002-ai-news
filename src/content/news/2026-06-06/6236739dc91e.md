# ERRORQUAKE: Heavy-Tailed Error Severity Distributions in Open-Weight Large Language Models
# ERRORQUAKE：开源大语言模型中重尾错误严重程度分布的研究

At matched accuracy, open-weight LLMs differ substantially in the shape of their error severity distribution -- a difference invisible to the scalar error rate. Hallucination benchmarks report a single error count and treat all errors as equivalent, yet a wrong date and a fabricated court ruling differ by orders of magnitude.

在准确率相当的情况下，开源大语言模型（LLM）在错误严重程度分布的形态上存在显著差异——这种差异是标量错误率所无法体现的。目前的幻觉基准测试通常只报告单一的错误计数，并将所有错误视为等同，然而，一个日期错误与一个捏造的法庭裁决在严重程度上有着数量级的差别。

We introduce Errorquake-10k, a 10,000-query benchmark scoring each response on a continuous 0-4 severity scale across 8 domains and 5 difficulty tiers, and we fit per-model severity distributions for 21 open-weight models. For each model we estimate a severity distribution index (b, the Gutenberg-Richter upper-tail slope) with 95% bootstrap confidence intervals.

我们推出了 Errorquake-10k，这是一个包含 10,000 个查询的基准测试，它在 8 个领域和 5 个难度等级上，以 0-4 的连续严重程度量表对每个回答进行评分，并为 21 个开源模型拟合了各自的严重程度分布。对于每个模型，我们通过 95% 自助法（bootstrap）置信区间估算了严重程度分布指数（b，即 Gutenberg-Richter 上尾斜率）。

Headline: across the 210 model pairs, 85 have disjoint 95% b confidence intervals at matched accuracy (|Delta epsilon| < 0.05) on human-consensus scoring, e.g. deepseek-v3.2 vs. ministral-14b at epsilon = 0.586 and Delta b = 0.47. A 519-item three-rater human validation study confirms measurement reliability (ICC(2,k=3) = 0.85), validates the LLM-judge ranking (rho = 0.89), and confirms the dense-model scaling correlation on human data (rho_s = -0.86).

核心发现：在 210 对模型组合中，有 85 对在人类共识评分且准确率匹配（|Δε| < 0.05）的情况下，其 95% 的 b 值置信区间互不重叠。例如，在 ε = 0.586 时，deepseek-v3.2 与 ministral-14b 的 Δb 达到 0.47。一项包含 519 个条目、由三名评估员参与的人类验证研究证实了测量可靠性（ICC(2,k=3) = 0.85），验证了 LLM 判别模型的排名准确性（ρ = 0.89），并确认了人类数据中密集模型缩放的相关性（ρ_s = -0.86）。

We prove a Non-Reducibility Theorem showing that severity profile and error rate are informationally non-redundant (I(b; model | epsilon) = 1.56 bits; 64.5% of cross-model b variance is unexplained by epsilon). A severity mechanism taxonomy (kappa = 0.83) reveals that error type shifts categorically with severity: low-severity errors are retrievals (71%); high-severity errors are fabrications (39%) -- and this composition differs by model size (p < 0.0001). Severity distribution should be reported alongside accuracy; it carries discriminative information that the error rate cannot.

我们证明了一个“不可约定理”（Non-Reducibility Theorem），表明严重程度分布与错误率在信息上是不可冗余的（I(b; model | ε) = 1.56 比特；跨模型 b 方差的 64.5% 无法由 ε 解释）。严重程度机制分类法（κ = 0.83）揭示了错误类型会随严重程度发生类别性转变：低严重程度错误多为检索错误（71%），而高严重程度错误多为捏造事实（39%）——且这种构成比例随模型规模的不同而存在差异（p < 0.0001）。因此，严重程度分布应与准确率一同报告；它携带了错误率所无法提供的判别性信息。
