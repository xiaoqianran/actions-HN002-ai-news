# NIV: Neural Axis Variations for Variable Font Generation
# NIV：用于可变字体生成的神经轴变体

Variable fonts enable continuous variation of glyph geometry along semantic design axes such as weight, width, slant, and optical size. However, constructing a variable font from a static font remains a labor-intensive process requiring expert typographic design and manual specification of glyph variation data.
可变字体允许字形几何形状沿着语义设计轴（如字重、字宽、倾斜度和视觉尺寸）进行连续变化。然而，从静态字体构建可变字体仍然是一个劳动密集型的过程，需要专业的排版设计和手动指定字形变化数据。

We introduce NIV (Neural Axis Variations), a method that automatically converts a static font into a fully functional variable font. Given glyph outlines and a set of desired design axes, NIV predicts per-point displacements. The model operates directly on vector glyph geometry and employs a novel Property Embedding mechanism that captures interactions between multiple axes, enabling consistent multi-axis variation within a unified framework.
我们引入了 NIV（Neural Axis Variations），这是一种将静态字体自动转换为功能齐全的可变字体的技术。给定字形轮廓和一组所需的设计轴，NIV 可以预测每个点的位移。该模型直接在矢量字形几何结构上运行，并采用了一种新颖的属性嵌入（Property Embedding）机制，能够捕捉多个轴之间的相互作用，从而在统一的框架内实现一致的多轴变化。

We train NIV on a newly constructed dataset derived from variable Google Fonts, comprising over one million variation tuples. The resulting model generalizes across unseen code points, unseen font styles, high-complexity CJK glyphs, and even out-of-distribution handwriting inputs. The generated outputs are standard variable font files supporting continuous interpolation via existing rendering engines.
我们在从 Google 可变字体中提取的新构建数据集上训练了 NIV，该数据集包含超过一百万个变化元组。由此产生的模型能够泛化到未见过的代码点、未见过的字体样式、高复杂度的中日韩（CJK）字形，甚至是分布外的手写输入。生成的输出是标准的变量字体文件，支持通过现有的渲染引擎进行连续插值。

To facilitate research, we release the dataset, the complete training and inference implementation, and trained models at this https URL. Beyond typography, our approach demonstrates how structured geometric objects with continuous parametric variation can be synthesized using neural deformations.
为了促进研究，我们在此链接发布了数据集、完整的训练和推理实现以及训练好的模型。除了排版领域之外，我们的方法还展示了如何利用神经变形技术合成具有连续参数变化的结构化几何对象。
