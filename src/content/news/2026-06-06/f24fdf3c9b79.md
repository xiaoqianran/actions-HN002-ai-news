# Differentiable Efficient Operator Search

**Title:** Differentiable Efficient Operator Search
**标题：** 可微高效算子搜索

**Abstract:** Efficient multimodal foundation models often rely on manually designed token-reduction operators, such as pruning, merging, pooling, and adaptive reweighting. Although these operators appear different, we show that they can be interpreted as distinct regimes of a shared operator space.
**摘要：** 高效多模态基础模型通常依赖于人工设计的 Token 缩减算子，例如剪枝、合并、池化和自适应重加权。尽管这些算子看起来各不相同，但我们证明了它们可以被解释为同一共享算子空间下的不同机制。

Based on this view, we introduce Efficient Operator Search, a differentiable framework that jointly searches where to reduce tokens, how many tokens to retain, and how reduced token information should be processed. The proposed search space parameterizes layer activation, retention budget, and operator behavior, while the search policy optimizes task performance under one-sided budget and cost constraints.
基于这一观点，我们引入了“高效算子搜索”（Efficient Operator Search），这是一个可微框架，能够联合搜索 Token 的缩减位置、保留数量以及缩减后的 Token 信息应如何处理。所提出的搜索空间对层激活、保留预算和算子行为进行了参数化，而搜索策略则在单侧预算和成本约束下优化任务性能。

This formulation recovers representative hand-designed baselines as special cases and further discovers hybrid operators beyond isolated manual designs. Experiments on multimodal benchmarks show that the searched operators achieve competitive accuracy-efficiency trade-offs, especially under aggressive visual-token reduction. These results suggest that efficient multimodal inference can be reframed from manual operator design to differentiable operator search.
该公式将具有代表性的手工设计基线作为特例进行了复现，并进一步发现了超越孤立人工设计的混合算子。在多模态基准测试上的实验表明，搜索出的算子实现了极具竞争力的精度-效率权衡，特别是在激进的视觉 Token 缩减场景下。这些结果表明，高效多模态推理可以从人工算子设计转向可微算子搜索。
