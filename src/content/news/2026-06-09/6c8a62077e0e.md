How It Works (Skills, Commands, Plugins)
Skills are the building blocks of the marketplace. Each skill gives Claude domain knowledge, analytical frameworks, or a guided workflow for a specific PM task. Some skills also work as reusable foundations that multiple commands share. Skills are loaded automatically when relevant to the conversation — no explicit invocation needed. If needed (e.g., prioritizing skills over general knowledge), you can force loading skills with /plugin-name:skill-name or /skill-name (Claude will add the prefix).
工作原理（技能、命令、插件）
“技能”（Skills）是该市场的基石。每一项技能都为 Claude 提供特定 PM 任务所需的领域知识、分析框架或引导式工作流。部分技能还可作为可复用的基础，供多个命令共享。当对话涉及相关内容时，技能会自动加载，无需显式调用。如果需要（例如优先使用特定技能而非通用知识），你可以通过 `/plugin-name:skill-name` 或 `/skill-name` 强制加载（Claude 会自动添加前缀）。

Commands are user-triggered workflows invoked with /command-name. They chain one or more skills into an end-to-end process. For example, /discover chains four skills together: brainstorm-ideas → identify-assumptions → prioritize-assumptions → brainstorm-experiments.
“命令”（Commands）是由用户通过 `/command-name` 触发的工作流。它们将一项或多项技能串联成端到端的流程。例如，`/discover` 将四项技能串联在一起：头脑风暴想法 → 识别假设 → 优先级排序 → 头脑风暴实验。

Plugins group related skills and commands into installable packages. Each plugin covers a PM domain — discovery, strategy, execution, and so on. Installing the marketplace gives you all 9 plugins at once.
“插件”（Plugins）将相关的技能和命令打包成可安装的组件。每个插件覆盖一个 PM 领域，如发现、战略、执行等。安装该市场即可一次性获得全部 9 个插件。

Commands use skills. Some skills serve multiple commands. Some skills (like prioritization-frameworks or opportunity-solution-tree) are standalone references that Claude draws on whenever relevant — no command needed. Commands are designed to flow into each other, matching the PM workflow. After any command completes, it suggests relevant next commands — just follow the prompts.
命令调用技能。有些技能服务于多个命令。有些技能（如优先级框架或机会解决方案树）是独立的参考资料，Claude 会在相关时自动调用，无需命令。命令的设计旨在相互衔接，匹配 PM 工作流。当任何命令完成后，它都会建议相关的后续命令，只需跟随提示操作即可。

Installation
安装指南

Claude Cowork (recommended for non-developers)
Open Customize (bottom-left)
Go to Browse plugins → Personal → +
Select Add marketplace from GitHub
Enter: phuryn/pm-skills
All 9 plugins install automatically. You get both commands (/discover, /strategy, etc.) and skills.
Claude Cowork（推荐非开发者使用）
打开“自定义”（左下角）
进入“浏览插件” → “个人” → “+”
选择“从 GitHub 添加市场”
输入：phuryn/pm-skills
全部 9 个插件将自动安装。你将同时获得命令（/discover, /strategy 等）和技能。

Claude Code (CLI)
# Step 1: Add the marketplace
claude plugin marketplace add phuryn/pm-skills
# Step 2: Install individual plugins
claude plugin install pm-toolkit@pm-skills
... (rest of the install commands)
Claude Code (CLI)
# 第一步：添加市场
claude plugin marketplace add phuryn/pm-skills
# 第二步：安装单个插件
claude plugin install pm-toolkit@pm-skills
...（其余安装命令）

Codex CLI (OpenAI)
Codex reads the same plugin marketplace file as Claude Code, so you can install PM Skills natively — no conversion or file-copying needed.
Codex CLI (OpenAI)
Codex 读取与 Claude Code 相同的插件市场文件，因此你可以原生安装 PM Skills，无需转换或复制文件。

Other AI assistants (skills only)
The skills/*/SKILL.md files follow the universal skill format and work with any tool that reads it. Commands (/slash-commands) are Claude-specific.
其他 AI 助手（仅限技能）
`skills/*/SKILL.md` 文件遵循通用技能格式，适用于任何支持该格式的工具。命令（/slash-commands）仅限 Claude 使用。
