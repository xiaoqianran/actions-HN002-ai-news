### Active usage of Claude Code hooks
### 积极使用 Claude Code 钩子

Claude Code Hooks is also a very interesting concept. Hooks are basically code that you can run at specific points in time. Some different hooks you have in Claude Code are:
* On startup of Claude Code
* On close of Claude Code
* Whenever the agent asks the user a question
* Whenever the agent finishes a task

Claude Code Hooks 也是一个非常有趣的概念。钩子本质上是在特定时间点运行的代码。Claude Code 中包含的一些钩子包括：
* Claude Code 启动时
* Claude Code 关闭时
* 代理向用户提问时
* 代理完成任务时

Basically, you can ensure a piece of code always runs whenever any of these events happen. To be specific, you can, for example, make Claude Code automatically generalize knowledge from the thread it’s been working on when you close it. Or you can make it ping you with a sound on your computer whenever it asks you a question and finishes a task.

基本上，你可以确保每当这些事件发生时，一段代码总会运行。具体来说，例如，你可以让 Claude Code 在关闭时自动总结它在当前线程中处理的知识。或者，你可以让它在提问或完成任务时，通过电脑发出提示音来提醒你。
