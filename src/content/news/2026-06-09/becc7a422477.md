# Direct 3D-Aware Object Insertion via Decomposed Visual Proxies
# 通过分解视觉代理实现直接的 3D 感知对象插入

**Abstract:** Object insertion aims to seamlessly composite a reference object into a specified region of a background image. Recent diffusion-based methods achieve high visual quality but formulate insertion as a simple 2D inpainting task, providing no explicit control over the object's 3D pose and limiting their practical applicability.

**摘要：** 对象插入旨在将参考对象无缝合成到背景图像的指定区域中。近期基于扩散模型的方法虽然实现了高质量的视觉效果，但通常将插入过程简化为 2D 图像修复任务，无法对对象的 3D 姿态进行显式控制，从而限制了其实际应用价值。

We propose DIRECT (Decomposed Injection for Reference Composition and Target-integration), a novel framework that integrates interactive pose manipulation with high-fidelity 2D image synthesis to enable pose-controllable object insertion.

我们提出了 DIRECT（用于参考合成与目标集成的分解注入框架），这是一个将交互式姿态操作与高保真 2D 图像合成相结合的新型框架，旨在实现可控姿态的对象插入。

Our method decomposes the insertion conditions into three complementary components: appearance guidance capturing visual details from the reference object, geometry guidance derived from the user-adjusted 3D proxy, and context guidance from the target background.

我们的方法将插入条件分解为三个互补的组件：从参考对象中捕获视觉细节的外观引导、从用户调整的 3D 代理中导出的几何引导，以及来自目标背景的上下文引导。

By injecting them through separate pathways, DIRECT avoids feature entanglement and simultaneously preserves reference appearance, follows the user-specified pose, and adapts the object to the target scene. We also introduce an automated data construction pipeline to improve the diversity and quality of training data. Experiments show that DIRECT outperforms previous methods in both geometric controllability and visual quality.

通过独立的路径注入这些信息，DIRECT 避免了特征纠缠，并能同时保持参考对象的外观、遵循用户指定的姿态，并将对象适配到目标场景中。我们还引入了一个自动化数据构建流水线，以提高训练数据的多样性和质量。实验表明，DIRECT 在几何可控性和视觉质量方面均优于以往的方法。
