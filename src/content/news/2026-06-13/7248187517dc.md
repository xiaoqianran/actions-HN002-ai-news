# ECA: Efficient Continual Alignment for Open-Ended Image-to-Text Generation
# ECA：面向开放式图文生成的高效持续对齐方法

**Abstract:** Incremental Learning (IL) for Open-ended Image-to-Text Generation (OpenITG) enables models to continuously generate accurate, contextually relevant text for new images while preserving previously acquired knowledge. 

**摘要：** 面向开放式图文生成（OpenITG）的增量学习（IL）使模型能够在处理新图像时持续生成准确且上下文相关的文本，同时保留先前习得的知识。

Unlike prior studies, this paper addresses a more practical scenario in which the predominant category of visual data shifts over time as environments evolve. In this context, we introduce a new notion of continual alignment, which incrementally adapts the alignment module within pre-trained VLMs to preserve high-quality cross-modal representations. 

与以往研究不同，本文探讨了一个更具实际意义的场景：随着环境的演变，视觉数据的主导类别会随时间发生偏移。在此背景下，我们引入了一种新的“持续对齐”概念，通过增量式调整预训练视觉语言模型（VLM）中的对齐模块，以保持高质量的跨模态表征。

Based on this idea, we propose Efficient Continual Alignment (ECA), a novel exemplar-free IL approach for OpenITG. The key challenge is enabling the model to acquire new, task-specific features while minimizing interference with the established alignment without accessing raw data from previous tasks. 

基于这一理念，我们提出了高效持续对齐（ECA），这是一种面向 OpenITG 的新型无样本增量学习方法。其核心挑战在于：如何在不访问先前任务原始数据的情况下，使模型获取新的任务特定特征，同时最大限度地减少对既有对齐机制的干扰。

To address this, ECA employs three core mechanisms: a Mixture of Query (MoQ) module that adapts task-specific query tokens, a Fisher Dynamic Expansion (FeDEx) that dynamically expands model structure based on a Fisher Information Matrix (FIM)-based metric, and an embedding dictionary with Dictionary Replay (DR) to retain past knowledge. 

为了解决这一问题，ECA 采用了三个核心机制：一是用于适配任务特定查询标记的查询混合（MoQ）模块；二是基于费舍尔信息矩阵（FIM）指标动态扩展模型结构的费舍尔动态扩展（FeDEx）；三是利用字典重放（DR）的嵌入字典，以保留历史知识。

To evaluate ECA's performance, we construct four new IL OpenITG benchmarks that better reflect real-world scenarios. Experimental results demonstrate that ECA significantly mitigates catastrophic forgetting and improves IL performance compared to baseline methods. Code and benchmarks are available at this https URL.

为了评估 ECA 的性能，我们构建了四个能够更好地反映真实世界场景的全新 IL OpenITG 基准测试。实验结果表明，与基准方法相比，ECA 显著缓解了灾难性遗忘，并提升了增量学习的性能。代码和基准测试已在指定链接发布。
