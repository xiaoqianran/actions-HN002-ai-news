# Microsoft PowerToys

Microsoft PowerToys is a collection of utilities that help you customize Windows and streamline everyday tasks.
Microsoft PowerToys 是一套实用工具集，旨在帮助用户自定义 Windows 系统并简化日常任务。

### 🔨 Utilities
### 🔨 实用工具

PowerToys includes over 30 utilities to help you customize and optimize your Windows experience: Advanced Paste, Always on Top, Awake, Color Picker, Command Not Found, Command Palette, Crop And Lock, Environment Variables, FancyZones, File Explorer Add-ons, File Locksmith, Grab And Move, Hosts File Editor, Image Resizer, Keyboard Manager, Light Switch, Mouse Utilities, Mouse Without Borders, New+, Peek, PowerDisplay, PowerRename, PowerToys Run, Quick Accent, Registry Preview, Screen Ruler, Shortcut Guide, Text Extractor, Workspaces, ZoomIt.
PowerToys 包含 30 多种实用工具，可帮助您自定义并优化 Windows 使用体验，包括：高级粘贴 (Advanced Paste)、始终置顶 (Always on Top)、唤醒 (Awake)、颜色选择器 (Color Picker)、命令未找到 (Command Not Found)、命令面板 (Command Palette)、裁剪与锁定 (Crop And Lock)、环境变量 (Environment Variables)、FancyZones、文件资源管理器插件 (File Explorer Add-ons)、文件锁匠 (File Locksmith)、抓取与移动 (Grab And Move)、Hosts 文件编辑器 (Hosts File Editor)、图像缩放器 (Image Resizer)、键盘管理器 (Keyboard Manager)、灯光开关 (Light Switch)、鼠标工具 (Mouse Utilities)、无界鼠标 (Mouse Without Borders)、New+、Peek、PowerDisplay、PowerRename、PowerToys Run、快速重音 (Quick Accent)、注册表预览 (Registry Preview)、屏幕标尺 (Screen Ruler)、快捷键指南 (Shortcut Guide)、文本提取器 (Text Extractor)、工作区 (Workspaces) 以及 ZoomIt。

### 📦 Installation
### 📦 安装

For detailed installation instructions and system requirements, visit the installation docs. But to get started quickly, choose one of the installation methods below:
如需详细的安装说明和系统要求，请访问安装文档。若想快速上手，请选择以下任一安装方式：

**Download the .exe file from GitHub**
Go to the PowerToys GitHub releases, scroll down and select Assets to reveal the installation files, and choose the one that matches your architecture and install scope. For most devices, that would be x64 per-user.
**从 GitHub 下载 .exe 文件**
前往 PowerToys 的 GitHub 发布页面，向下滚动并选择“Assets”以展开安装文件，选择与您的架构和安装范围相匹配的文件。对于大多数设备，通常是 x64 用户安装版 (per-user)。

**Microsoft Store**
You can easily install PowerToys from the Microsoft Store.
**Microsoft Store**
您可以直接从 Microsoft Store 轻松安装 PowerToys。

**WinGet**
Download PowerToys from [WinGet](https://github.com/microsoft/winget-cli#installing-the-client). Updating PowerToys via winget will respect the current PowerToys installation scope. To install PowerToys, run the following command from the command line / PowerShell:
**WinGet**
通过 [WinGet](https://github.com/microsoft/winget-cli#installing-the-client) 下载 PowerToys。使用 winget 更新 PowerToys 时，将遵循当前的安装范围。若要安装 PowerToys，请在命令行或 PowerShell 中运行以下命令：

*User scope installer (default):* `winget install Microsoft.PowerToys -s winget`
*用户范围安装程序（默认）：* `winget install Microsoft.PowerToys -s winget`

*Machine-wide scope installer:* `winget install --scope machine Microsoft.PowerToys -s winget`
*机器范围安装程序：* `winget install --scope machine Microsoft.PowerToys -s winget`

**Other methods**
There are [community driven install methods](https://learn.microsoft.com/windows/powertoys/install#community-driven-install-tools) such as Chocolatey and Scoop. If these are your preferred install solutions, you can find the install instructions there.
**其他方法**
此外还有 [社区驱动的安装方式](https://learn.microsoft.com/windows/powertoys/install#community-driven-install-tools)，例如 Chocolatey 和 Scoop。如果您更倾向于这些安装方案，可以在相应页面找到安装说明。
