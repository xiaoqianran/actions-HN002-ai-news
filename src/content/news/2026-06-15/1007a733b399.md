# GorvGoyl / Clone-Wars

**Clone Wars - Open source clones of popular sites**
**Clone Wars - 热门网站的开源克隆项目**

100+ open-source clones and alternatives of popular sites like Airbnb, Amazon, Instagram, Netflix, TikTok, Spotify, WhatsApp, YouTube, etc. List contains source code, tutorials, demo links, tech stack, and GitHub stars count. Great for learning purpose! -Made by Gourav Goyal

Two kinds of projects on this list:
此列表包含两类项目：

*   **Clones:** look quite similar (UI-wise) but aren't fully-functional, mostly made for learning purposes.
    *   **克隆版 (Clones)：** 外观非常相似（UI 层面），但功能不全，主要用于学习目的。
*   **Alternatives:** fully-functional open-source alternatives of popular software.
    *   **替代品 (Alternatives)：** 热门软件的功能完备的开源替代方案。

Seeing GitHub stars will give you a rough idea about which one is which.
查看 GitHub 星标数可以让你大致了解哪些是克隆版，哪些是替代品。
