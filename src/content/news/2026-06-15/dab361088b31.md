# Telegram Stars Economics for Bot Developers: What Your Stars Are Actually Worth in 2026
# Telegram Stars 经济学：开发者指南——你的 Stars 在 2026 年到底值多少钱？

Your Telegram bot is live. Users are sending Stars. You check your creator dashboard and see 847 Stars sitting there. 你的 Telegram 机器人已经上线，用户正在发送 Stars。当你查看创作者仪表板时，看到账户里有 847 个 Stars。随之而来的问题总是同一个：这折合成真实货币到底是多少钱？答案比 Telegram 界面显示的要复杂得多——如果你正在构建一个通过 Stars 变现的机器人或小程序，在发布定价之前，你必须先理解其中的差额。

### The Fundamental Problem: Stars Have Two Different Values
### 核心问题：Stars 具有两种不同的价值

Telegram Stars are not a fixed currency. There are at least two distinct values that matter:
Telegram Stars 并非固定货币。至少有两种不同的价值需要关注：

| Context | Who pays it | What it represents |
| :--- | :--- | :--- |
| **Buyer price** | Your user in the Telegram app | What they pay to acquire Stars (includes app store markup) |
| **Creator withdrawal value** | You, the developer | What you actually receive after Telegram's share, TON conversion, and exchange fees |

| 上下文 | 支付方 | 代表含义 |
| :--- | :--- | :--- |
| **买家价格** | Telegram 应用内的用户 | 他们获取 Stars 所支付的金额（包含应用商店抽成） |
| **创作者提现价值** | 你（开发者） | 在扣除 Telegram 抽成、TON 转换及交易所费用后，你实际收到的金额 |

These two numbers are not the same. And neither is displayed prominently in Telegram's UI.
这两个数字并不相等，且 Telegram 的界面中也没有显著显示它们。

### Why the Withdrawal Path Matters
### 为什么提现路径很重要

Stars don't convert directly to USD or EUR. The flow is: Stars earned → eligible for withdrawal (minimum threshold + waiting period) → TON (at Telegram's conversion rate) → TON wallet or exchange → sell TON for USDT/USD/EUR → fiat to bank (KYC may apply).
Stars 无法直接转换为美元或欧元。其流程为：赚取 Stars → 达到提现资格（最低门槛 + 等待期）→ 转换为 TON（按 Telegram 汇率）→ 进入 TON 钱包或交易所 → 将 TON 出售为 USDT/USD/EUR → 法币转入银行（可能需要 KYC）。

Each step has friction:
每一步都有摩擦成本：
* Waiting period before Stars are eligible for withdrawal. (Stars 获得提现资格前的等待期)
* Minimum amount before you can withdraw. (提现前的最低金额限制)
* TON price volatility between earning and selling. (赚取到出售期间的 TON 价格波动)
* Exchange fees when converting TON to fiat. (将 TON 转换为法币时的交易所费用)
* KYC at the fiat off-ramp if you haven't set it up. (如果尚未设置，法币出金时需要 KYC)

For small amounts this is fine. For a subscription-based bot with recurring Star payments, you want to model this before you set your pricing.
对于小额收入，这没问题。但对于采用订阅制且有持续 Stars 支付的机器人，你在设定定价前必须先对这些成本进行建模。

### Fragment: The Cheaper Buying Option
### Fragment：更便宜的购买选项

Your users can buy Stars cheaper than the in-app price using Fragment, Telegram's official TON marketplace. Fragment removes the mobile app-store markup. Users need TON and a compatible wallet, which adds setup time — but for regular buyers or larger amounts, the savings are meaningful.
用户可以使用 Telegram 官方的 TON 市场 Fragment 以低于应用内的价格购买 Stars。Fragment 移除了移动应用商店的抽成。用户需要准备 TON 和兼容的钱包，这会增加设置时间，但对于经常购买的用户或大额购买而言，节省的费用非常可观。

What this means for your bot: a user who buys Stars through Fragment pays less per Star than a user buying through the Telegram app. Your revenue per Star doesn't change — but your users' cost does. Knowing this exists is useful if users ask why Stars are expensive.
这对你的机器人意味着：通过 Fragment 购买 Stars 的用户，其单价低于通过 Telegram 应用内购买的用户。你每个 Star 的收入不会改变，但用户的成本降低了。如果用户询问为什么 Stars 很贵，了解这一点会很有帮助。

### FAQ for Developers
### 开发者常见问题解答

**Q: Can I withdraw Stars immediately after a user sends them?**
**问：用户发送 Stars 后，我可以立即提现吗？**
A: Not immediately. Stars go through an eligibility period before you can withdraw. Check the current Telegram/Fragment rules before planning cash flow.
答：不能立即提现。Stars 在你可以提现之前会经过一个资格审查期。在规划现金流之前，请查看当前的 Telegram/Fragment 规则。

**Q: Do Stars expire?**
**问：Stars 会过期吗？**
A: Earned Stars don't expire, but check current Telegram terms before building long-term assumptions on this.
答：赚取的 Stars 不会过期，但在基于此建立长期假设之前，请查看当前的 Telegram 条款。

**Q: Can I use Stars as an in-app currency inside my mini app?**
**问：我可以在小程序内将 Stars 作为应用内货币使用吗？**
A: Yes — this is a core use case. Stars can be used for in-app purchases within Telegram mini apps via the Telegram Payments API.
答：可以，这是核心用例。通过 Telegram 支付 API，Stars 可用于 Telegram 小程序内的应用内购买。

**Q: Is there a way to test the Stars payment flow in development?**
**问：在开发过程中有办法测试 Stars 支付流程吗？**
A: Telegram's Bot API supports test mode for payments. Check the official Telegram Bot API docs for `sendInvoice` and the Stars payment flow.
答：Telegram Bot API 支持支付测试模式。请查阅官方 Telegram Bot API 文档中关于 `sendInvoice` 和 Stars 支付流程的部分。

**Q: What's the minimum withdrawal amount?**
**问：最低提现金额是多少？**
A: Telegram and Fragment set minimums that can change. Check the current docs before planning your cash-out schedule.
答：Telegram 和 Fragment 设定的最低限额可能会变动。在规划提现计划前，请查阅最新文档。
