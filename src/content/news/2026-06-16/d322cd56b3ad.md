# Hetzner Price Adjustment / Hetzner 价格调整

### Limited offerings / 限时/限量供应
The availability of these limited offerings is dependent on the hardware used in these servers. We will only offer the limited pricing tier once servers with a significantly reduced procurement cost are available in our data center. This can sometimes take up to several weeks for a meaningful number of servers to be available (mostly dependent on cancellations) and they will be offered as long as supply lasts.
这些限量供应产品的可用性取决于服务器所使用的硬件。我们仅在数据中心有采购成本显著降低的服务器时，才会提供此限价档位。有时可能需要数周时间才能有足够数量的服务器可用（主要取决于退订情况），且售完即止。

Please note that limited offerings still contain all the service and support of a regular server type. We strongly recommend to also look into our server auction for some great deals on older server models. Prices in the server auction are defined by offer and demand based on a Dutch-style auction. Read more in our Server Auction FAQs.
请注意，限量供应产品仍包含常规服务器类型的所有服务与支持。我们强烈建议您查看我们的服务器拍卖页面，以获取旧款服务器的优惠。拍卖价格由供需关系决定，采用荷兰式拍卖方式。详情请参阅我们的服务器拍卖常见问题解答。
