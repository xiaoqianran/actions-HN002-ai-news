# continuedev / continue

**Continue: Pioneering open-source coding agent**
**Continue：开创性的开源编程智能体**

**What is Continue?**
**什么是 Continue？**

Note: The continuedev/continue repository is no longer actively maintained and is read-only for all users.
注意：continuedev/continue 仓库已不再进行主动维护，目前对所有用户设为只读状态。

Continue is a coding agent available as a CLI, VS Code extension, and JetBrains plugin.
Continue 是一款编程智能体，提供 CLI 工具、VS Code 扩展以及 JetBrains 插件。

**Documentation**
**文档**

To learn how to configure Continue, how it works, and how to customize it, check out the Continue Docs.
如需了解如何配置 Continue、其工作原理以及如何进行自定义，请查看 Continue 文档。

**Final 2.0.0 Release**
**最终 2.0.0 版本发布**

We polished Continue and did a final 2.0.0 release of the VS Code extension, CLI, and JetBrains plugin. This included removing anonymous telemetry, pulling out authentication, squashing bugs, and more.
我们对 Continue 进行了完善，并发布了 VS Code 扩展、CLI 和 JetBrains 插件的最终 2.0.0 版本。此次更新包括移除匿名遥测数据、剔除身份验证功能、修复漏洞等。
