### OmniIDE 1.0.4 Released
### OmniIDE 1.0.4 版本发布

OmniNodeCo has released OmniIDE 1.0.4 on GitHub. This release introduces several improvements aimed at making setup and day-to-day development smoother:
OmniNodeCo 已在 GitHub 上发布了 OmniIDE 1.0.4 版本。此次更新引入了多项改进，旨在使安装配置和日常开发流程更加顺畅：

**What's New**
**更新内容**

Documentation website – New online documentation to help users get started and find information more easily.
文档网站 – 全新的在线文档，旨在帮助用户更轻松地入门并查找所需信息。

Git auto-install prompt – OmniIDE can now detect when Git is not available and prompt users to install it.
Git 自动安装提示 – OmniIDE 现在可以检测系统是否未安装 Git，并提示用户进行安装。

Improved stability – Multiple fixes and refinements to improve reliability and overall user experience.
稳定性提升 – 进行了多项修复和优化，以提高软件的可靠性和整体用户体验。

Release GitHub: V1.0.4
GitHub 发布地址：V1.0.4

OmniIDE is an open-source IDE developed by OmniNodeCo. Feedback, bug reports, and contributions are welcome as the project continues to evolve.
OmniIDE 是由 OmniNodeCo 开发的一款开源 IDE。随着项目的持续演进，我们欢迎用户提供反馈、提交错误报告以及参与贡献。
