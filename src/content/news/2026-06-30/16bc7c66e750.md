# Unified Zero-Shot Time Series Forecasting: A Darts Foundation
# 统一零样本时间序列预测：Darts 基础架构

**Abstract:** Since its initial release in 2020, Darts has become a widely used open-source Python library for time series analysis. A series of foundation models have recently claimed accuracy improvements in zero-shot forecasting, promising a paradigm shift from training custom models to harnessing pre-trained general-purpose forecasters.

**摘要：** 自 2020 年首次发布以来，Darts 已成为广泛使用的时间序列分析开源 Python 库。近期，一系列基础模型声称在零样本预测方面提升了准确性，预示着从“训练定制模型”到“利用预训练通用预测器”的范式转变。

Foundation models, however, are often released as isolated packages with fragmented interfaces and limited interoperability with common tooling, making joint evaluation and integration within complete pipelines difficult.

然而，基础模型通常以独立软件包的形式发布，其接口碎片化且与常用工具的互操作性有限，这使得在完整工作流中进行联合评估和集成变得十分困难。

In Darts, we developed a unified `FoundationModel` class collection (Chronos-2, TimesFM 2.5, TiRex, PatchTST-FM) that provides standardized, full-cycle forecasting interfaces with minimal external dependencies for integrating foundation models into the ecosystem.

在 Darts 中，我们开发了一套统一的 `FoundationModel` 类集合（包含 Chronos-2、TimesFM 2.5、TiRex 和 PatchTST-FM），提供了标准化的全周期预测接口，并以极少的外部依赖将这些基础模型集成到生态系统中。

Existing Darts pipelines can now use foundation models with only a name change; new pipelines can use them for zero-shot or fine-tuned forecasting, uncertainty estimation, and backtesting, combined with data processing and evaluation tooling, all within a unified framework.

现有的 Darts 工作流现在只需更改名称即可使用这些基础模型；新的工作流则可以将它们用于零样本或微调预测、不确定性估计和回测，并结合数据处理与评估工具，所有这些功能均可在统一的框架内实现。
