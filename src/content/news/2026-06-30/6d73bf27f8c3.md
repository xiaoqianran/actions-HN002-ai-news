# SelectAnyTree: A Promptable Instance Segmentation Model for 3D Forest LiDAR Point Clouds
# SelectAnyTree：一种用于森林三维激光雷达点云的可提示实例分割模型

Automated instance segmentation of forest LiDAR point clouds is increasingly critical as forest monitoring moves toward scalable, detailed, 3D measurement. Yet, progress is constrained by label scarcity for tree instances; a single hectare can hold millions of points and hundreds of overlapping, complex crowns, making manual annotation from scratch with raw data laborious and error-prone.
随着森林监测向可扩展、精细化的三维测量方向发展，森林激光雷达（LiDAR）点云的自动化实例分割变得愈发重要。然而，树木实例标注数据的匮乏限制了该领域的发展；单公顷森林可能包含数百万个点和数百个重叠、复杂的树冠，这使得从原始数据开始进行人工标注既费力又容易出错。

Annotations are often corrected from automatic pre-segmentations, but remain costly as these provide no interactive or AI-assisted refinement. Inspired by the promptable paradigm of foundation segmentation models, we propose SelectAnyTree, a promptable instance segmentation model that delineates any individual tree in a 3D forest point cloud from a few clicks.
标注工作通常是在自动预分割的基础上进行修正，但由于缺乏交互式或人工智能辅助的优化手段，这种方法依然成本高昂。受基础分割模型“可提示（promptable）”范式的启发，我们提出了 SelectAnyTree，这是一种可提示的实例分割模型，只需几次点击即可勾勒出三维森林点云中的任意单棵树木。

It introduces two key components: Click-to-query prompt encoder and Canopy Height Model (CHM)-guided first prompt. The former turns each click into a single content query, encoding its 3D position and positive/negative polarity together with a pooled local backbone feature. The latter provides treetops as a geometry- and ecologically guided first prompt without any user input.
该模型引入了两个关键组件：点击查询提示编码器（Click-to-query prompt encoder）和冠层高度模型（CHM）引导的首个提示。前者将每次点击转化为单一的内容查询，将其三维位置、正/负极性与池化的局部骨干特征进行编码；后者则在无需用户输入的情况下，提供树梢位置作为几何和生态引导的首个提示。

The resulting prompt query is then decoded into one tree mask by a state-space query decoder to efficiently capture long-range context in large-scale forest scenes with linear-time complexity. We evaluate SelectAnyTree in interactive and instance-level settings across seven diverse forest regions and an independent held-out test dataset, demonstrating strong generalization beyond the training domains.
随后，生成的提示查询通过状态空间查询解码器（state-space query decoder）解码为单个树木掩码，从而以线性时间复杂度高效捕获大规模森林场景中的长距离上下文信息。我们在七个不同的森林区域及一个独立的留出测试数据集上，对 SelectAnyTree 在交互式和实例级设置下进行了评估，证明了其在训练域之外具有强大的泛化能力。

It segments a target tree to 78.2 Intersection over Union (IoU) from a single click, 24.8 points above the strongest promptable baseline, and reaches every accuracy target with the fewest clicks, while using far fewer parameters and less inference time than prior promptable models. The source code is available at this https URL.
该模型仅需单次点击即可将目标树木的分割交并比（IoU）达到 78.2，比目前最强的可提示基准模型高出 24.8 个百分点，并以最少的点击次数达到了各项精度指标。同时，与先前的可提示模型相比，它使用了更少的参数，推理时间也更短。源代码已在指定链接发布。
