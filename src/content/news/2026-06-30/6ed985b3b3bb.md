# Loko Scheme 0.13.0

Loko Scheme 0.13.0 is now available from: https://scheme.fail/releases/loko-0.13.0.tar.gz https://scheme.fail/releases/loko-0.13.0.tar.gz.sig
Loko Scheme 0.13.0 现已发布，下载地址：https://scheme.fail/releases/loko-0.13.0.tar.gz 以及签名文件 https://scheme.fail/releases/loko-0.13.0.tar.gz.sig

A bootable disk image for 64-bit PCs is available from: https://scheme.fail/releases/disk-images/loko-hdd-0.13.0.img.gz https://scheme.fail/releases/disk-images/loko-hdd-0.13.0.img.gz.sig
适用于 64 位 PC 的可启动磁盘镜像可从以下地址获取：https://scheme.fail/releases/disk-images/loko-hdd-0.13.0.img.gz 以及签名文件 https://scheme.fail/releases/disk-images/loko-hdd-0.13.0.img.gz.sig

The signatures are made with the GnuPG key 0xDD839B748F10AD4D.
上述签名均使用 GnuPG 密钥 0xDD839B748F10AD4D 生成。

Loko Scheme 0.13.0 fixes bugs, improves performance and adds features. See NEWS.md in the distribution for a more detailed summary of changes.
Loko Scheme 0.13.0 修复了若干错误，提升了性能并增加了新功能。有关更改的详细摘要，请参阅发行包中的 NEWS.md 文件。

Loko Scheme is an optimizing Scheme compiler that builds statically linked binaries for bare metal, Linux and NetBSD/amd64. It supports the R6RS Scheme and R7RS Scheme standards.
Loko Scheme 是一款优化型 Scheme 编译器，可为裸机（bare metal）、Linux 和 NetBSD/amd64 构建静态链接的二进制文件。它支持 R6RS 和 R7RS Scheme 标准。

Loko Scheme’s web site is https://scheme.fail, where you can find the release tarballs and the manual. There is also a mailing list at https://lists.scheme.fail.
Loko Scheme 的官方网站是 https://scheme.fail，您可以在该网站找到发布包和手册。此外，项目还设有邮件列表：https://lists.scheme.fail。
