# Cursor now has a mobile app for guiding your coding agent on the go
# Cursor 现已推出移动端应用，助你随时随地指挥编程智能体

Cursor isn’t letting the $60 billion SpaceX acquisition slow it down. On Monday, the company announced a new app called Cursor Mobile, designed for users who want to prompt coding agents directly from their phone.
尽管面临 600 亿美元的 SpaceX 收购案，Cursor 的发展步伐并未放缓。周一，该公司宣布推出一款名为 Cursor Mobile 的新应用，旨在让用户能够直接通过手机向编程智能体发送指令。

The app ties into the Cursor 2.0 changes unveiled in October, which shifted the service toward independent coding agents. With the mobile app, users can spin up new coding agents or interact with agents that were initiated from the desktop client.
该应用与 10 月份发布的 Cursor 2.0 更新紧密结合，后者将服务重心转向了独立的编程智能体。通过这款移动应用，用户可以创建新的编程智能体，或与在桌面端启动的智能体进行交互。

Cursor’s move to mobile follows similar apps from Anthropic and OpenAI, which both offer ways to interact with their coding tools on mobile. It’s part of a broader shift in AI-based coding tools, which are increasingly abstracting away from written code and toward oversight of code-writing agents.
Cursor 此举紧随 Anthropic 和 OpenAI 的步伐，这两家公司此前都已推出了各自的移动端编程工具交互方式。这是 AI 编程工具领域更广泛趋势的一部分：开发重心正逐渐从编写代码本身，转向对代码编写智能体的监督与管理。

With no need to access large code bases, many developers are switching away from multi-monitor desktop setups in favor of phones, which allow continuous conversations with remote agents.
由于无需直接访问庞大的代码库，许多开发者正逐渐放弃多显示器的桌面配置，转而使用手机，因为手机能够实现与远程智能体的持续对话。

In a recent talk, Anthropic’s head of Claude Code Boris Cherny said he had almost entirely switched to mobile AI coding as a result. “Most of my coding now is on my phone,” Cherny said in the talk. “I would have said ‘you’re crazy’ if you told me that six months ago, but yeah, here we are.”
在最近的一次演讲中，Anthropic 旗下 Claude Code 的负责人 Boris Cherny 表示，他已经几乎完全转向了移动端 AI 编程。“我现在大部分的编程工作都是在手机上完成的，”Cherny 在演讲中说道，“如果六个月前有人这么跟我说，我会觉得他疯了，但事实就是这样。”
