# Memory-Augmented LSTM Autoencoder for Unsupervised Activity Recognition with IMU Sensor Fusion
# 基于记忆增强型 LSTM 自编码器与 IMU 传感器融合的无监督活动识别

**Abstract:** HAR using Inertial Measurement Unit (IMU) sensors is vital for healthcare monitoring and rehabilitation. Despite deep learning advancements, major challenges remain: reliance on labeled data, multi-sensor fusion complexity, and the limited ability of unsupervised methods to capture spatiotemporal dependencies. These issues are pronounced in real-world scenarios with noisy data, overlapping activities, and missing labels.

**摘要：** 利用惯性测量单元（IMU）传感器进行人体活动识别（HAR）对于医疗监测和康复至关重要。尽管深度学习取得了长足进步，但仍面临诸多挑战：对标注数据的依赖、多传感器融合的复杂性，以及无监督方法在捕捉时空依赖性方面的能力有限。在存在噪声数据、活动重叠和标签缺失的现实场景中，这些问题尤为突出。

We propose a fully unsupervised spatiotemporal feature fusion framework using a memory-augmented autoencoder. It enhances activity representations via short temporal windows of multi-sensor IMU data, enabling real-time applications. Our framework extracts hierarchical static features via a Stacked Autoencoder, fusing them within and across sensors. A sequence-to-sequence LSTM Autoencoder then temporally refines these features, incorporating historical motion patterns without labels.

我们提出了一种使用记忆增强型自编码器的全无监督时空特征融合框架。该框架通过多传感器 IMU 数据的短时间窗口增强活动表征，从而实现实时应用。我们的框架通过堆叠自编码器（Stacked Autoencoder）提取分层静态特征，并在传感器内部及跨传感器进行融合。随后，序列到序列（Seq2Seq）的 LSTM 自编码器对这些特征进行时间维度的精炼，在无需标签的情况下整合历史运动模式。

We analyze key hyperparameters to identify configurations that maximize feature separability under short-window constraints. Evaluated on DaLiAc and PAMAP2 using realistic inter-class window segmentation, our method achieves 96.6% and 98.4% accuracy, respectively, surpassing supervised baselines and unsupervised approaches. Our method improves feature separability by up to 9% despite shorter temporal windows. While our realistic inter-class segmentation reduces accuracy by ~7%, it was intentionally adopted to better reflect real-world activity transitions and practical relevance.

我们分析了关键超参数，以确定在短窗口约束下最大化特征可分性的配置。在 DaLiAc 和 PAMAP2 数据集上使用真实的类间窗口分割进行评估，我们的方法分别达到了 96.6% 和 98.4% 的准确率，超越了监督学习基准和现有的无监督方法。尽管采用了更短的时间窗口，我们的方法仍将特征可分性提高了 9%。虽然真实的类间分割使准确率降低了约 7%，但这是为了更好地反映现实世界中的活动转换和实际应用价值而特意采用的。
