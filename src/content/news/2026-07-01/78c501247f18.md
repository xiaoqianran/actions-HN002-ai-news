# Claude Science is Anthropic’s newest flagship product
# Claude Science 是 Anthropic 最新的旗舰产品

**EXECUTIVE SUMMARY** At an event for pharmaceutical executives, biotech founders, and researchers on Tuesday, Anthropic announced Claude Science, a major new product intended to support scientific research in the same way that Claude Code supports software engineering.
**执行摘要** 周二，在面向制药高管、生物技术创始人和研究人员举办的活动上，Anthropic 发布了 Claude Science。这是一款重磅新产品，旨在以 Claude Code 支持软件工程的方式，为科学研究提供支持。

Like Claude Code, Claude Science can autonomously carry out meaningful work when given concise, high-level instructions, and it has access to tools that make it particularly useful for research in computational biology and drug development. Along with launching and previewing Claude Science, which is now available to all paid Claude subscribers, Anthropic also announced that it will be using the product to pursue some of its own research into drugs for rare, neglected diseases.
与 Claude Code 一样，Claude Science 在接收到简洁的高层指令后，能够自主开展有意义的工作。它还可以调用各种工具，使其在计算生物学和药物开发研究中尤为实用。除了发布并预览 Claude Science（现已向所有 Claude 付费订阅用户开放）之外，Anthropic 还宣布将利用该产品开展针对罕见病和被忽视疾病的药物研究。

This is not Anthropic’s first foray into AI for science. In October, the company released plug-ins that help Claude make use of scientific software and databases under the heading “Claude for Life Sciences.” But unlike this earlier release, Claude Science is a full-featured, standalone product.
这并非 Anthropic 首次涉足科学 AI 领域。去年 10 月，该公司曾发布插件，帮助 Claude 使用科学软件和数据库，并将其归类为“Claude for Life Sciences”。但与之前的发布不同，Claude Science 是一款功能齐全的独立产品。

Anthropic’s decision to elevate Claude Science to the same rank as Claude Code and Claude Cowork indicates that the company is taking AI’s scientific applications very seriously—or at least wants to give the impression that it is. “It represents how important this is to our mission that this is right up there with Claude Code and Claude Cowork as the next really significant product that we’re releasing,” says Eric Kauderer-Abrams, Anthropic’s head of life sciences. “Our mission is to develop AI that serves humanity’s long-term well-being, and we believe that by far the greatest opportunity to do that is in the life sciences.”
Anthropic 决定将 Claude Science 提升至与 Claude Code 和 Claude Cowork 同等地位，表明该公司非常重视 AI 的科学应用——或者至少希望外界认为如此。“这代表了它对我们使命的重要性，它与 Claude Code 和 Claude Cowork 并列，是我们发布的下一个真正重要的产品，”Anthropic 生命科学主管 Eric Kauderer-Abrams 表示，“我们的使命是开发服务于人类长期福祉的 AI，我们认为实现这一目标最大的机遇就在生命科学领域。”

For the past decade, one company—Google DeepMind—has been at the vanguard of AI for science. CEO Demis Hassabis and researcher John Jumper won the Nobel Prize in chemistry for their work on the company’s AlphaFold model, and DeepMind has also made major contributions to meteorology, materials science, and a variety of other disciplines. But in the past several months, the fast-advancing frontier of AI progress seems to have left DeepMind in the dust. When it comes to coding, which has become the most lucrative use case for LLMs, DeepMind is stuck playing catch-up.
在过去十年中，Google DeepMind 一直处于科学 AI 的前沿。首席执行官 Demis Hassabis 和研究员 John Jumper 因在 AlphaFold 模型上的工作获得了诺贝尔化学奖，DeepMind 在气象学、材料科学和其他多个学科也做出了重大贡献。但在过去几个月里，AI 进步的飞速发展似乎让 DeepMind 逐渐落后。在编码这一大语言模型（LLM）最赚钱的应用场景上，DeepMind 陷入了追赶者的角色。

Anthropic is well positioned to take up DeepMind’s scientific mantle. Like Hassabis, Anthropic CEO Dario Amodei is a PhD scientist—unlike OpenAI CEO Sam Altman, who’s a businessman through and through. Many scientists are already avid users of tools such as Claude Code. These days, a lot of scientific research involves some amount of coding, but not all scientists are expert software engineers, and so tools like Claude Code can make a huge difference for their productivity. And the company has recently earned a major scientific vote of confidence: Earlier this month, Jumper announced that he is leaving DeepMind for Anthropic.
Anthropic 完全有能力接过 DeepMind 在科学领域的衣钵。与 Hassabis 一样，Anthropic 首席执行官 Dario Amodei 是一位拥有博士学位的科学家，这与纯粹的商人 OpenAI 首席执行官 Sam Altman 不同。许多科学家已经是 Claude Code 等工具的忠实用户。如今，大量的科学研究涉及一定程度的编码，但并非所有科学家都是专业的软件工程师，因此像 Claude Code 这样的工具可以极大地提升他们的生产力。此外，该公司最近还获得了一项重大的科学信任投票：本月初，Jumper 宣布离开 DeepMind 加入 Anthropic。

Since agents powered by LLMs, including Anthropic’s Opus model series, became capable of useful, independent work in late 2025, scientists have been seeing just how much they can do. In a blog post published on Anthropic’s website, the Harvard physicist Matthew Schwartz estimated, on the basis of his work with Claude Code and other Anthropic tools, that the company’s Opus 4.5 model is about as capable of executing scientific projects as a second-year graduate student.
自 2025 年末由 LLM（包括 Anthropic 的 Opus 模型系列）驱动的智能体具备了进行有效、独立工作的能力以来，科学家们已经见识到了它们的强大潜力。哈佛大学物理学家 Matthew Schwartz 在 Anthropic 官网发表的一篇博文中估计，基于他使用 Claude Code 和其他 Anthropic 工具的工作经验，该公司的 Opus 4.5 模型在执行科学项目方面的能力大约相当于一名二年级研究生。

According to Kauderer-Abrams, Claude Science isn’t intended to displace Claude Code and Claude Cowork in scientists’ workflows. Instead, it’s designed to build on what scientists already find useful about Anthropic’s products. For instance, it not only writes code but also helps scientists run their code on powerful computer clusters, which many scientists need for their work but can be difficult to manage. And it prioritizes reproducibility, so that scientists can trace back the source of any figure or result and check it for accuracy and validity.
据 Kauderer-Abrams 称，Claude Science 并非旨在取代科学家工作流程中的 Claude Code 和 Claude Cowork。相反，它的设计初衷是在科学家认为 Anthropic 产品有用的基础上进行扩展。例如，它不仅能编写代码，还能帮助科学家在强大的计算机集群上运行代码——许多科学家在工作中需要用到这些集群，但管理起来却很困难。此外，它还优先考虑可重复性，以便科学家能够追溯任何图表或结果的来源，并检查其准确性和有效性。

Though Claude Science could in principle assist with any area of scientific research, it seems designed and marketed as a tool for molecular and cellular biology, and for drug development in particular. It can interface with various tools used in genetics, chemistry, and protein biology, all of which could come in handy for researchers on the hunt for new drugs. During the Tuesday event, Alexander Tarashansky, who led the development of Claude Science, demonstrated how the system could autonomously identify new drug candidates for phenylketonuria, a rare genetic disease.
虽然 Claude Science 原则上可以辅助任何科学研究领域，但它似乎被设计并营销为分子和细胞生物学，特别是药物开发领域的工具。它可以与遗传学、化学和蛋白质生物学中使用的各种工具对接，这些对于寻找新药的研究人员来说非常方便。在周二的活动中，领导 Claude Science 开发的 Alexander Tarashansky 演示了该系统如何自主识别苯丙酮尿症（一种罕见遗传病）的新候选药物。

And Anthropic isn’t leaving all that work to the pharma companies and university labs that were represented at the event. Armed with Claude Science, it will be pursuing its own research into drug candidates for neglected diseases—both to help move science forward and to gain a clearer sense of how Claude Science works in the real world.
Anthropic 并没有将所有工作都留给活动中出现的制药公司和大学实验室。凭借 Claude Science，该公司将开展针对被忽视疾病的候选药物研究，这既是为了推动科学进步，也是为了更清晰地了解 Claude Science 在现实世界中的运作方式。

There are obvious humanitarian reasons to prioritize drug development when creating a general-purpose scientific research tool, and AI industry leaders often cite curing disease as a major potential upside of the technology. But it’s also notable that pharmaceutical companies have far deeper pockets than academic researchers. Anthropic says it’s set to see its first profitable quarter, and if major new contracts with pharmaceutical companies are forthcoming, they could help ensure it stays profitable as the tokenmaxxing craze dies down—something that’s ever more important as an IPO approaches later this year.
在创建通用科学研究工具时，优先考虑药物开发有着明显的人道主义理由，AI 行业领袖也经常将治愈疾病作为该技术的一大潜在优势。但同样值得注意的是，制药公司的财力远比学术研究人员雄厚。Anthropic 表示，公司即将迎来首个盈利季度。如果能与制药公司达成重大的新合同，将有助于确保其在“代币消耗（tokenmaxxing）”热潮消退后保持盈利——随着今年晚些时候 IPO 的临近，这一点变得愈发重要。
