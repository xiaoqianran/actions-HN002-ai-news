# Pidgin 3.0 Alpha 2 (2.96.0) has been released

We are ecstatic to announce that we have released the second **alpha** version of Pidgin 3.0!!! This release has an official version number of 2.96.0. We will continue releasing in this fashion until we are ready for 3.0.0 which will be the release we consider ready for end users. You can read more about our development phases [here](https://discourse.imfreedom.org/t/pidgin-3-development-phases/369).

我们非常激动地宣布，Pidgin 3.0 的第二个 **Alpha** 版本现已发布！！！该版本的官方版本号为 2.96.0。我们将继续以这种方式进行发布，直到我们准备好 3.0.0 版本，即我们认为适合最终用户使用的版本。您可以[点击此处](https://discourse.imfreedom.org/t/pidgin-3-development-phases/369)了解更多关于我们开发阶段的信息。

The release can be found on [SourceForge](https://sourceforge.net/projects/pidgin/files/Pidgin3/2.96.0/) and should show up on [Flathub Beta](https://docs.flathub.org/docs/for-users/installation) shortly.

该版本可在 [SourceForge](https://sourceforge.net/projects/pidgin/files/Pidgin3/2.96.0/) 上找到，并很快会在 [Flathub Beta](https://docs.flathub.org/docs/for-users/installation) 上架。

If you don’t want to use the Flathub Beta method, there are build instructions in the `README.md` for building a Flatpak locally or installing into a `meson devenv`. Both methods avoid installing directly to your OS installation which is the preferred method as there is no easy way to uninstall the application at this time.

如果您不想使用 Flathub Beta 方法，`README.md` 中提供了在本地构建 Flatpak 或安装到 `meson devenv` 的构建说明。这两种方法都避免了直接安装到您的操作系统中，这是目前推荐的做法，因为现阶段还没有简便的方法来卸载该应用程序。

## About

## 关于

Just like the previous releases, this is a timed release. It is a snapshot of what we currently have and is something we think other tinkerers and developers may want to start playing with, but again, it is **not** meant for *end users*.

与之前的版本一样，这是一个定时发布版本。它是我们当前开发进度的快照，我们认为其他技术爱好者和开发者可能会想尝试一下，但再次强调，它**不**适合*最终用户*使用。

Most things are not implemented, the user interface has not been polished at all, and there are so, so many bugs. As such we are asking that packagers please **do not package** this for your users yet as the potential support requests will be too much for us to handle at this time.

大多数功能尚未实现，用户界面也完全没有经过润色，并且存在非常多的 Bug。因此，我们恳请软件包维护者暂时**不要**为您的用户打包此版本，因为目前我们无法处理由此产生的海量支持请求。

Looking forward, **Alpha 3** is scheduled for **2026-09-30** and will have a version number of **2.97.0**.

展望未来，**Alpha 3** 计划于 **2026 年 9 月 30 日**发布，版本号为 **2.97.0**。

We have a bunch of tickets entered for Alpha 3 and you can keep an eye on the [Burn down Chart for Alpha 3](https://issues.imfreedom.org/reports/burndown/165-5) to stay informed.

我们已经为 Alpha 3 创建了许多工单，您可以关注 [Alpha 3 的燃尽图 (Burn down Chart)](https://issues.imfreedom.org/reports/burndown/165-5) 以了解最新进展。

As always, priorities and possibilities can and will change, so even if something is listed for this release that doesn’t mean it will be in the release until it’s actually completed.

一如既往，优先级和可能性可能会发生变化，因此即使某项功能被列入此版本的计划中，也不代表它一定会包含在最终发布版本里，除非它已实际完成。

If you would like monthly updates, be sure to check out the [state-of-the-bird](https://discourse.imfreedom.org/tag/state-of-the-bird) posts!

如果您想获取每月更新，请务必查看 [state-of-the-bird](https://discourse.imfreedom.org/tag/state-of-the-bird) 系列文章！

## Notifications

## 通知

We removed the purple-toast and notification-sound plugins and built support for them directly into Pidgin. You can configure everything from the preferences with additional settings.

我们移除了 purple-toast 和 notification-sound 插件，并将它们的支持直接内置到了 Pidgin 中。您可以在首选项中通过附加设置来配置所有内容。
