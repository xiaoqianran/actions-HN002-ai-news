# IonSense-QKG: A Quantum-Readiness Metadata Framework for Lithium-Ion Battery Dataset Discovery
# IonSense-QKG：用于锂离子电池数据集发现的量子就绪元数据框架

Public lithium-ion battery datasets are increasingly used for state-of-health estimation, remaining-useful-life prediction, anomaly detection, electrochemical diagnostics, second-life analytics, and battery safety research. However, these datasets vary substantially in chemistry, modality, scale, label quality, sequence structure, access status, and preprocessing complexity. These differences directly affect whether a dataset is feasible for near-term hybrid quantum-classical machine-learning workflows.

公共锂离子电池数据集正越来越多地被用于健康状态评估、剩余使用寿命预测、异常检测、电化学诊断、梯次利用分析以及电池安全研究。然而，这些数据集在化学成分、模态、规模、标签质量、序列结构、访问状态和预处理复杂度方面存在显著差异。这些差异直接影响了数据集是否适用于近期的混合量子-经典机器学习工作流。

This paper presents IonSense-QKG, a quantum-readiness metadata framework for lithium-ion battery dataset discovery. Starting from the EV-Battery-IonSense index, the proposed framework enriches public battery dataset records with quantum-relevant metadata, including task type, sensing modality, chemistry, label availability, sequence type, preprocessing requirements, candidate quantum encodings, estimated qubit range, and NISQ feasibility.

本文提出了 IonSense-QKG，这是一个用于锂离子电池数据集发现的量子就绪元数据框架。该框架以 EV-Battery-IonSense 索引为基础，通过量子相关元数据丰富了公共电池数据集记录，包括任务类型、传感模态、化学成分、标签可用性、序列类型、预处理要求、候选量子编码、估计量子比特范围以及 NISQ（含噪声中等规模量子）可行性。

A transparent Quantum Readiness Score is introduced to rank datasets as candidate resources for future hybrid quantum-classical battery benchmarks. The score is intended as a dataset-selection heuristic, not as evidence of quantum advantage. The framework demonstrates query-based discovery over enriched metadata to identify datasets suitable for compact quantum feature maps, quantum time-series workflows, limited-label anomaly detection, and future battery-health benchmarking.

文中引入了一个透明的“量子就绪评分”（Quantum Readiness Score），用于对数据集进行排名，将其作为未来混合量子-经典电池基准测试的候选资源。该评分旨在作为数据集选择的启发式指标，而非量子优势的证明。该框架展示了如何通过对丰富后的元数据进行基于查询的发现，从而识别出适用于紧凑型量子特征映射、量子时间序列工作流、有限标签异常检测以及未来电池健康基准测试的数据集。

The released artifact includes metadata tables, scoring scripts, robustness checks, link-checking utilities, and SQL-style query examples. IonSense-QKG positions dataset selection as a data-management problem and provides a reproducible foundation for data-centric quantum battery analytics.

发布的成果包括元数据表、评分脚本、稳健性检查、链接检查工具以及 SQL 风格的查询示例。IonSense-QKG 将数据集选择定位为一个数据管理问题，并为以数据为中心的量子电池分析提供了可复现的基础。
