# CanvasAgent: Enabling Complex Image Creation and Editing via Visual Tool Orchestration
# CanvasAgent：通过视觉工具编排实现复杂的图像创作与编辑

Complex image creation and editing often require more than a single generation or editing model. A user request may involve synthesizing images, localizing objects, segmenting regions, editing selected content, compositing intermediate assets, reading text, and enhancing the final result. Such tasks shift multimodal agents from perception-augmented reasoning to manipulation-centered visual creation, where tools must actively transform visual states rather than merely inspect them.

复杂的图像创作与编辑往往不仅仅依赖于单一的生成或编辑模型。用户的需求可能涉及图像合成、目标定位、区域分割、选定内容编辑、中间资产合成、文本读取以及最终效果增强。这类任务将多模态智能体从“感知增强推理”转向了“以操作为中心的视觉创作”，在这一过程中，工具必须主动改变视觉状态，而不仅仅是进行观察。

However, existing multimodal tool-use agents are mostly optimized for perception, search, or domain-specific editing, and lack large-scale supervision for executable image-creation trajectories. In this paper, we introduce CanvasCraft, a large-scale multimodal tool-use dataset for complex image creation and editing, and CanvasAgent, a tool-augmented multimodal agent that learns to orchestrate heterogeneous visual tools through multi-turn interaction.

然而，现有的多模态工具使用智能体大多针对感知、搜索或特定领域的编辑进行了优化，缺乏用于可执行图像创作轨迹的大规模监督数据。在本文中，我们引入了 CanvasCraft，这是一个用于复杂图像创作与编辑的大规模多模态工具使用数据集；同时提出了 CanvasAgent，这是一种工具增强型多模态智能体，它通过多轮交互学习如何编排异构视觉工具。

CanvasCraft contains 140K fully annotated executable trajectories and 10K RL task specifications. CanvasAgent is first trained with SFT to learn executable reasoning-action trajectories, and is then optimized with GRPO using a hybrid reward that combines outcome- and process-level signals. During rollout, CanvasAgent inspects intermediate results, tracks visual assets, and adapts tool decisions to the evolving visual state.

CanvasCraft 包含 14 万条完全标注的可执行轨迹和 1 万个强化学习（RL）任务规范。CanvasAgent 首先通过监督微调（SFT）进行训练，以学习可执行的推理-行动轨迹，随后使用结合了结果级和过程级信号的混合奖励，通过组相对策略优化（GRPO）进行优化。在执行过程中，CanvasAgent 会检查中间结果、跟踪视觉资产，并根据不断变化的视觉状态调整工具决策。

Experiments evaluate both final image quality and trajectory behavior, demonstrating the effectiveness of CanvasAgent and the proposed dataset for complex multi-tool image creation workflows.

实验评估了最终图像质量和轨迹行为，证明了 CanvasAgent 和所提出的数据集在复杂的多工具图像创作工作流中的有效性。
