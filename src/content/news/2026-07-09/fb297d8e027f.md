# Granger Causal Networks and Indirect Feedback
# 格兰杰因果网络与间接反馈

**A non-parametric variable selection for Structural VARs**
**一种用于结构化向量自回归（SVAR）的非参数变量选择方法**

One of the most utilized econometric workflows in the last decade has been that of using vector autoregressive models. From research done by academicians to economists informing policy implementation have all utilized VAR models in some shape or iteration [think vector error correcting models(VECM) or Structural VARs (SVAR)]. It could’ve been for impulse response studies, endogenous variable forecasts or something as simple as arguing cross-correlation between temporal variables.
在过去十年中，最常用的计量经济学工作流程之一是使用向量自回归（VAR）模型。从学术研究到经济学家为政策实施提供参考，各方都在不同程度上使用了 VAR 模型（例如向量误差修正模型 VECM 或结构化向量自回归 SVAR）。这些模型常用于脉冲响应研究、内生变量预测，或是简单的论证时间变量之间的互相关性。

Unfortunately, one problem that VAR studies have not been able to solve for is breaking out the impact of one endogenous variable at a point in time on another into direct, indirect and aggregate feedback. Intuitively, this can be understood by the idea that if one variable affects another directly then we can connect the two explicitly in an arbitrary equation: X is Structurally Impacting Y. But if the variable only has an intermediate feedback onto another variable, we can connect them implicitly: X only affects Y through it’s impact on Z.
遗憾的是，VAR 研究一直未能解决的一个问题是：如何将一个内生变量在某一时间点对另一个变量的影响，拆解为直接反馈、间接反馈和总反馈。直观地理解，如果一个变量直接影响另一个变量，我们可以在方程中显式地连接两者：X 在结构上影响 Y。但如果该变量仅通过中间变量产生反馈，我们则可以隐式地连接它们：X 仅通过影响 Z 来影响 Y。

Although we are able to measure aggregate impact through orthogonal impulse-response functions, we cannot still break it out into indirect and direct feedback since that would require us to trace effects through EACH equation in the VAR model. This is a computational nightmare! [Readers can be tempted to use SEM models but they also require us to a-priori define relationships within variables and then see how well the data fits our hypothesis; the problem we’re trying to solve here is “what are the relationships in the first place?”]
尽管我们能够通过正交脉冲响应函数来衡量总影响，但仍无法将其拆解为间接反馈和直接反馈，因为这需要追踪 VAR 模型中“每一个”方程的影响。这简直是计算上的噩梦！（读者可能会倾向于使用结构方程模型 SEM，但它们同样要求我们先验地定义变量间的关系，再观察数据对假设的拟合程度；而我们这里试图解决的问题是：“这些关系最初究竟是什么？”）

This article introduces a little talked about topic within the econometrics community: Causality Network Graphs. The idea of a causal graph is simple: if a variable, A, causes variable B then we visually draw an edge going from A -> B. We do this for all pairs of variables in the dataset. The direction of the causality matters as well. A can cause B but the opposite can also be true. The graph formed using the set of variables/nodes and edges is called a causality network graph, G(e,d). Where e is the number of edges and d is the number of vertices (variables) in the dataset. For computational purposes we represent G(e,d) using an adjacency matrix.
本文介绍了一个在计量经济学界鲜少被讨论的话题：因果网络图。因果图的概念很简单：如果变量 A 导致变量 B，我们就画一条从 A 指向 B 的边。我们对数据集中的所有变量对都进行此操作。因果的方向也很重要，A 可能导致 B，反之亦然。由变量/节点集合和边组成的图被称为因果网络图 G(e,d)，其中 e 是边的数量，d 是数据集中的顶点（变量）数量。为了便于计算，我们使用邻接矩阵来表示 G(e,d)。

Causality network graphs become important in panel data research as a way to visualize variables and how they are connected to others. As a result, the graphs become easy candidates for quick and inexpensive inference measurement. Here, we’re going to attempt to try and build a G(e,d) using real life data and also explore, in layman’s terms, how these networks can help us begin to understand direct and indirect effects between 2 endogenous variables.
因果网络图在面板数据研究中变得非常重要，它提供了一种可视化变量及其相互连接方式的方法。因此，这些图表成为了进行快速且低成本推断测量的理想工具。在这里，我们将尝试使用真实数据构建一个 G(e,d)，并以通俗易懂的方式探讨这些网络如何帮助我们理解两个内生变量之间的直接和间接影响。

For our reproducible example, let’s look at log returns of an ETF that tracks the NASDAQ-100 index, QQQ. Additionally, we will also look some technical indicators like Relative Strength Index (RSI), Bollinger Percent B (pctB), Volume, Range, and prices of another security – SPY and see if network graphs can help us understand how structurally causal these indicators are to the day-to-day price movements of QQQ, if at all. We’re going to suitably transform these variables to ensure stationarity.
为了进行可重复的示例，我们来看看追踪纳斯达克 100 指数的 ETF（QQQ）的对数收益率。此外，我们还将观察一些技术指标，如相对强弱指数（RSI）、布林带百分比（pctB）、成交量、波动幅度以及另一只证券 SPY 的价格，看看网络图是否能帮助我们理解这些指标对 QQQ 日常价格变动的结构性因果影响（如果有的话）。我们将对这些变量进行适当的转换以确保平稳性。

The novelty of G(e,d) comes from the rules we use to build it. One of the easiest ways to build G(e,d) could be to look at the correlations between these variables and only select highly correlated (cor > |0.6|) edges to make the graph. G(e,d) 的新颖之处在于我们构建它时所使用的规则。构建 G(e,d) 最简单的方法之一是观察变量间的相关性，并仅选择高度相关（相关系数 > |0.6|）的边来绘制图形。以下是使用此规则构建的网络图输出：

It appears as though all the variables are connected to others in both directions. 2-way connectedness may impart information on which variables are connected but not how they are related. For network graphs using correlation coefficients as the rule to build graph edges, we don’t need a directed graph given the implied symmetry. Unfortunately, the graph does not give us any additional insight. We know correlation does not equal causation. This was a test and numbers agree.
看起来所有变量在两个方向上都与其他变量相连。双向连接可能传达了哪些变量是相关的，但无法说明它们是如何相关的。对于使用相关系数作为构建边规则的网络图，由于其隐含的对称性，我们不需要有向图。遗憾的是，该图并没有给我们提供额外的见解。我们知道相关性不等于因果关系，这只是一个测试，数据结果也印证了这一点。

Next, let’s use pairwise granger causality to build a bivariate Granger Causal Network Graph. Here the direction matters and we can build a directed network. For G(e,d), each edge indicates whether the lags of the independent variable are, up to a given level of significance, important in explaining variation in the dependent variable on top of its own lags.
接下来，让我们使用成对格兰杰因果关系来构建双变量格兰杰因果网络图。在这里，方向至关重要，我们可以构建一个有向网络。对于 G(e,d)，每一条边都表示在给定显著性水平下，自变量的滞后项在解释因变量变动时，是否比因变量自身的滞后项更具重要性。

Now we’re getting somewhere. Although RSI seemed to be very weakly correlated to QQQ log returns, it has a 99% significant pairwise granger causal effect on QQQ log returns. QQQ log returns seem to have aggregate causal implications from: RSI, Range, SPY log returns, pctB.
现在我们有了一些进展。尽管 RSI 与 QQQ 对数收益率的相关性似乎非常微弱，但它对 QQQ 对数收益率却具有 99% 显著性的成对格兰杰因果影响。QQQ 的对数收益率似乎受到以下因素的总因果影响：RSI、波动幅度、SPY 对数收益率以及 pctB。

Note that QQQ log returns are not directly causally related to QQQ Volume. However, if we consider the chain – Volume > Range > QQQ log returns, we might say that volume has an intermediate causal implication on QQQ log returns as well. What about other variables? Below are some paths we can take from RSI to QQQ log returns:
请注意，QQQ 对数收益率与 QQQ 成交量之间没有直接的因果关系。然而，如果我们考虑链条——成交量 > 波动幅度 > QQQ 对数收益率，我们或许可以说成交量对 QQQ 对数收益率也具有间接的因果影响。那么其他变量呢？以下是从 RSI 到 QQQ 对数收益率的一些路径：

* RSI > SPY.diff > QQQ.diff [intermediate / 中间]

Remember that this list is not exhaustive. There are multiple other paths leading from RSI to QQQ log returns.
