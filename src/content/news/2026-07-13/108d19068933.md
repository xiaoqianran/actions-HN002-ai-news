# 🚀 I Built CODELIVLY — A Platform to Learn Cybersecurity Through Hands-On Practice
# 🚀 我开发了 CODELIVLY —— 一个通过实战学习网络安全的平台

Over the past year, I've been building CODELIVLY, a cybersecurity learning platform focused on helping beginners and aspiring professionals learn by doing instead of just watching videos. 🔗 Website: https://codelivly.com
在过去的一年里，我一直在开发 CODELIVLY，这是一个专注于帮助初学者和有志于从事网络安全行业的专业人士通过“实战”而非仅仅观看视频来学习的平台。🔗 网站：https://codelivly.com

### Why I Started
### 我为何开启这个项目

When I began learning cybersecurity, I found myself jumping between YouTube tutorials, blogs, PDFs, labs, and documentation. Everything was scattered. I wanted one place where someone could: Learn the fundamentals, Practice real concepts, Track their progress, Earn certificates, and Engage with a learning community. That idea eventually became CODELIVLY.
当我刚开始学习网络安全时，我发现自己总是在 YouTube 教程、博客、PDF、实验环境和文档之间来回切换，所有资源都非常零散。我想要一个能让人们在一个地方完成以下所有事情的平台：学习基础知识、练习真实概念、跟踪学习进度、获得证书以及参与学习社区。这个想法最终演变成了 CODELIVLY。

### What the Platform Includes
### 平台包含的内容

The goal wasn't to build another course website. Instead, I wanted to create an ecosystem for continuous learning. Current features include: 📚 Structured learning paths, 🎯 Career-focused roadmaps, 🧪 Hands-on labs, 📝 Interactive quizzes, 🏆 Certificates, 👥 Community discussions, 🎮 Competitions, 📖 Cybersecurity glossary, 🛠️ Free cybersecurity resources and tools. Everything is designed to help learners progress from beginner concepts toward practical cybersecurity skills.
我的目标不是再建一个普通的课程网站，而是创建一个持续学习的生态系统。目前的功能包括：📚 结构化学习路径、🎯 职业导向路线图、🧪 实战实验室、📝 互动测验、🏆 证书、👥 社区讨论、🎮 竞赛、📖 网络安全术语表、🛠️ 免费网络安全资源与工具。一切设计旨在帮助学习者从基础概念进阶到实用的网络安全技能。

### Tech Stack
### 技术栈

The platform is built using a modern web stack, including: Next.js, React, TypeScript, Node.js, PostgreSQL, Prisma, and Tailwind CSS. A lot of effort also went into the admin dashboard for managing content, users, certificates, learning paths, analytics, and community features.
该平台使用现代 Web 技术栈构建，包括：Next.js、React、TypeScript、Node.js、PostgreSQL、Prisma 和 Tailwind CSS。此外，我在管理后台投入了大量精力，用于管理内容、用户、证书、学习路径、数据分析和社区功能。

### What I Learned
### 我的收获

Building the platform taught me that software isn't just about writing code. It's also about: Designing intuitive user experiences, Structuring content effectively, Handling authentication and permissions, Building scalable admin tools, and Listening to user feedback and iterating. The product has evolved significantly since the first version thanks to community feedback.
开发这个平台让我明白，软件开发不仅仅是写代码，还包括：设计直观的用户体验、有效地组织内容、处理身份验证与权限、构建可扩展的管理工具，以及倾听用户反馈并进行迭代。得益于社区的反馈，该产品自第一个版本以来已经有了显著的进化。

### What's Next
### 未来规划

I'm continuing to expand the platform with: More practical labs, Additional career paths, More community features, Better learning analytics, Improved mobile experience, and New free cybersecurity tools.
我将继续扩展平台，包括：更多的实战实验室、更多的职业路径、更多的社区功能、更好的学习分析、优化的移动端体验以及新的免费网络安全工具。
