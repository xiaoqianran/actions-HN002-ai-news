# YimMenu / YimMenuV2

**YimMenuV2 Experimental menu for GTA 5: Enhanced**
**YimMenuV2：GTA 5 增强版实验性菜单**

**How to use**
**如何使用**

Download the latest version of FSL from here and place version.dll in your GTA V directory. Using FSL is now optional but highly recommended for account safety.
从此处下载最新版本的 FSL，并将 version.dll 放入你的 GTA V 根目录。目前使用 FSL 是可选的，但为了账号安全，强烈建议使用。

Download YimMenuV2 from GitHub Releases.
从 GitHub Releases 下载 YimMenuV2。

Download an injector, such as Xenos.
下载一个注入器，例如 Xenos。

Open Rockstar Launcher, select Grand Theft Auto V Enhanced, go to settings, and disable BattlEye. If you are using Steam or Epic Games, you may have to pass the -nobattleye command line parameter as well.
打开 Rockstar Launcher，选择《侠盗猎车手 5 增强版》，进入设置并禁用 BattlEye。如果你使用的是 Steam 或 Epic Games 版本，可能还需要添加 `-nobattleye` 命令行参数。

Launch GTA V, then use your injector to inject YimMenuV2.dll at the main menu.
启动 GTA V，然后在主菜单界面使用注入器注入 YimMenuV2.dll。

**How to open the menu?**
**如何打开菜单？**

Press the INSERT key or Ctrl+\ to open the menu.
按下 INSERT 键或 Ctrl+\ 即可打开菜单。

**Common issues**
**常见问题**

**I keep getting desynced from public sessions every five minutes.**
**我每五分钟就会从公共战局中掉线（不同步）。**

We currently do not have a BattlEye bypass, and legitimate hosts will eventually remove you due to a heartbeat failure. There is currently no way to stop this other than using an actual (private) bypass.
我们目前没有 BattlEye 的绕过方案，正版主机最终会因为心跳检测失败而将你踢出。除了使用真正的（私有）绕过工具外，目前没有其他方法可以解决这个问题。

**I removed FSL and all my progress disappeared!**
**我移除了 FSL，结果所有进度都消失了！**

FSL reroutes account save data to disk, so any progress made with FSL will only show up if you have FSL enabled. If you don't want this, you can also use YimMenuV2 without FSL, but this is not recommended.
FSL 会将账号存档数据重定向到磁盘，因此任何在使用 FSL 时产生的进度，只有在开启 FSL 的情况下才会显示。如果你不希望这样，也可以在不使用 FSL 的情况下运行 YimMenuV2，但不建议这样做。

**I removed FSL and the game doesn't start up anymore.**
**我移除了 FSL，游戏无法启动了。**

This is a known issue; delete "Documents/GTAV Enhanced/Profiles" to fix.
这是一个已知问题；删除“Documents/GTAV Enhanced/Profiles”文件夹即可修复。
