# Apache Ossie

Apache Ossie (incubating) is a collaborative, open-source effort dedicated to standardizing and streamlining semantic model exchange and utilization across the diverse array of tools and platforms within the data analytics, AI, and BI ecosystem.
Apache Ossie (孵化中) 是一项协作型开源计划，致力于在数据分析、人工智能和商业智能 (BI) 生态系统中的各类工具与平台之间，实现语义模型交换与利用的标准化和精简化。

Our shared vision is to establish a common, vendor-agnostic semantic model specification, promoting unparalleled interoperability, efficiency, and collaboration among all participants.
我们的共同愿景是建立一套通用的、与供应商无关的语义模型规范，从而促进所有参与者之间实现无与伦比的互操作性、效率和协作。

By providing a single, consistent source of truth, this vendor-agnostic standard ensures that your data's definitions and value remain consistent as they are interchanged between AI agents, BI platforms, and all other tools in your ecosystem, eliminating inconsistencies across your different tools.
通过提供单一且一致的“事实来源”，这一与供应商无关的标准确保了数据在 AI 智能体、BI 平台以及生态系统中其他工具之间交换时，其定义和价值始终保持一致，从而消除了不同工具间的差异。

Apache Ossie was formerly known as Open Semantic Interchange (OSI).
Apache Ossie 前身为 Open Semantic Interchange (OSI)。

Apache Ossie provides a single JSON- and YAML-based specification that any tool can read and write, addressing the semantic fragmentation common across today's data stack: the same KPI defined differently across tools, teams spending significant effort manually reconciling definitions, and AI agents producing unreliable outputs grounded in inconsistent business logic.
Apache Ossie 提供了一套基于 JSON 和 YAML 的统一规范，任何工具均可对其进行读写。这解决了当今数据技术栈中常见的语义碎片化问题，例如：同一 KPI 在不同工具中定义各异、团队耗费大量精力手动协调定义，以及 AI 智能体因业务逻辑不一致而产生不可靠的输出。

### What's in this repository
### 仓库内容

*   **core-spec/** — The Ossie core specification (spec.md), the machine-readable schema (spec.yaml, osi-schema.json), and accompanying documentation.
    **core-spec/** — Ossie 核心规范 (spec.md)、机器可读的模式文件 (spec.yaml, osi-schema.json) 及相关文档。
*   **converters/** — Reference converters that translate between Ossie and other semantic formats (e.g., dbt, GoodData, Polaris, Salesforce).
    **converters/** — 用于在 Ossie 与其他语义格式（如 dbt、GoodData、Polaris、Salesforce）之间进行转换的参考转换器。
*   **examples/** — Example semantic models, including a complete TPC-DS model.
    **examples/** — 语义模型示例，包括一个完整的 TPC-DS 模型。
*   **validation/** — Tooling for validating semantic models against the Ossie schema.
    **validation/** — 用于根据 Ossie 模式验证语义模型的工具。
*   **docs/** — Project documentation and overview.
    **docs/** — 项目文档与概览。
