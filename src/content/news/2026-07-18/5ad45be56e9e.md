# Low-Latency Relay Selection in NR-V2X Vehicular Communications via Graph Isomorphism Networks with Edge Features
# 基于带边特征图同构网络的 NR-V2X 车联网低延迟中继选择

**Abstract:** Reliable, low-latency uplink connectivity is a key requirement for C-V2X networks in dense urban environments, where fast channel variations and blockages often degrade direct vehicle-to-infrastructure links. Multi-hop relaying can restore coverage, but relay-link activation under radio, capacity, and routing constraints results in an NP-hard optimisation problem, typically solved via Mixed-Integer Linear Programming (MILP), whose runtime scales poorly with graph size.

**摘要：** 在高密度城市环境中，可靠且低延迟的上行链路连接是 C-V2X 网络的核心需求，但快速的信道变化和遮挡往往会削弱车辆到基础设施（V2I）的直接链路。多跳中继技术可以恢复覆盖范围，但在无线电、容量和路由约束下的中继链路激活是一个 NP-hard 优化问题，通常通过混合整数线性规划（MILP）来解决，其运行时间会随图规模的增大而急剧增加。

This paper introduces an edge-aware Learning-to-Optimise framework for real-time relay selection. Each V2X snapshot is modelled as a directed graph: node features encode vehicle state and traffic demand, while edge features capture radio-link capacity. An offline MILP oracle generates optimal relay configurations that supervise a Graph Isomorphism Network with Edge Features (GINE), enabling edge-level relay activation through a single forward pass, with tightly bounded inference latency.

本文引入了一种用于实时中继选择的边缘感知“学习优化”（Learning-to-Optimise）框架。每个 V2X 快照都被建模为一个有向图：节点特征编码车辆状态和交通需求，而边特征则捕获无线链路容量。离线 MILP 预言机生成最优中继配置，用于监督带边特征的图同构网络（GINE），从而通过单次前向传播实现边缘级的中继激活，并具有严格受限的推理延迟。

To bridge learning and exact optimisation, we also propose a hybrid GINE-Pruned MILP (GP-MILP) strategy in which GINE predictions prune the MILP search space. Experiments on a large-scale dataset generated via an OSM-SUMO-GEMV$^2$ pipeline show that GINE closely matches MILP decisions at the link level (accuracy 0.9589, F1-score 0.9544 on validation) and yields consistent end-to-end connectivity gains over a 1-hop MILP baseline (up to 9.2% with four RSUs and 12% with two RSUs).

为了连接学习与精确优化，我们还提出了一种混合 GINE 剪枝 MILP（GP-MILP）策略，利用 GINE 的预测结果来剪枝 MILP 的搜索空间。通过 OSM-SUMO-GEMV$^2$ 流水线生成的大规模数据集实验表明，GINE 在链路层面的决策与 MILP 高度吻合（准确率为 0.9589，验证集 F1 分数为 0.9544），并且在端到端连接性方面比单跳 MILP 基准方案有持续提升（在 4 个 RSU 时提升高达 9.2%，在 2 个 RSU 时提升 12%）。

Inference latency remains tightly bounded, with all evaluated instances completing within 5 ms. Moreover, GP-MILP preserves MILP-equivalent solutions (same objective value) while achieving solver runtimes below 30 ms for more than 98% of the graph instances, making MILP-grade optimisation compatible with stringent NR-V2X latency budgets.

推理延迟保持在严格范围内，所有评估实例均在 5 毫秒内完成。此外，GP-MILP 在保留与 MILP 等效解（相同的目标函数值）的同时，超过 98% 的图实例求解时间低于 30 毫秒，使得 MILP 级别的优化能够满足 NR-V2X 严格的延迟预算要求。
