### UI Skills: Design Engineers' Toolkit

**UI Skills: Skills for Design Engineers**
UI Skills：设计工程师的技能库

**More on ui-skills.com**
更多信息请访问 ui-skills.com

**Run `npx ui-skills start` to route your agent through the right UI skill set for the task.**
运行 `npx ui-skills start`，即可引导你的 AI Agent 选择最适合当前任务的 UI 技能集。

**CLI**
命令行工具 (CLI)

*   `npx ui-skills`
*   `npx ui-skills start`
*   `npx ui-skills categories`
*   `npx ui-skills list --category motion`
*   `npx ui-skills get baseline-ui`
