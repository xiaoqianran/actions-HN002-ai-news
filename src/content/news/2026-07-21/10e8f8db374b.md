# Moonshine：让你通过PC向任何运行Moonlight的设备串流游戏

**Content / 内容：**

Moonshine 🌙 Moonshine lets you stream games from your PC to any device running Moonlight. Your keyboard, mouse, and controller inputs are sent back to the host so you can play games remotely as if you were sitting in front of it.

Moonshine 🌙 Moonshine 可让你将PC上的游戏串流到任何运行 Moonlight 的设备。你的键盘、鼠标和手柄输入会被发送回主机，让你可以远程玩游戏，就像坐在电脑前一样。

**Features / 功能特性：**

Isolated streaming sessions: Each stream runs in its own compositor, completely separate from your desktop environment. Your host PC can still be used for other things while you stream.

隔离串流会话：每个串流都在其独立的合成器中运行，与你的桌面环境完全分离。当你串流时，主机PC仍可用于其他任务。

No monitor required: Works on headless servers — no HDMI dummy plug needed.

无需显示器：可在无头服务器上运行——无需HDMI假负载。

Hardware video encoding: H.264, H.265, and AV1 encoding using the GPU.

硬件视频编码：使用GPU进行H.264、H.265和AV1编码。

⚠️ AV1 Warning: AV1 encoding is experimental and has issues on NVIDIA GPUs that cause frame sizes to grow over time (see issue). This should be fixed in driver version 595.44.3.0. Until then, stick with H.264 or H.265.

⚠️ AV1警告：AV1编码处于实验阶段，在NVIDIA GPU上存在问题，会导致帧大小随时间增长（参见问题）。这应在驱动程序版本595.44.3.0中修复。在此之前，请坚持使用H.264或H.265。

HDR support: True 10-bit HDR streaming for supported games.

HDR支持：为受支持的游戏提供真正的10位HDR串流。

Full input support: Mouse, keyboard, and gamepad (including motion, touchpad, and haptics).

完整输入支持：鼠标、键盘和手柄（包括动作、触摸板和触觉反馈）。

Audio streaming: Stereo and surround sound (5.1/7.1) with low-latency Opus encoding.

音频串流：立体声和环绕声（5.1/7.1），采用低延迟Opus编码。

**Requirements / 系统要求：**

Linux only. Tested on Arch Linux, but it's been reported to work on other Linux distributions too.

仅限Linux。已在Arch Linux上测试，但据报道也可在其他Linux发行版上运行。

systemd. Required for launching and managing application processes. Almost all modern Linux distributions include it by default.

systemd。用于启动和管理应用程序进程所必需。几乎所有现代Linux发行版都默认包含它。

A GPU with Vulkan video encoding. NVIDIA RTX, AMD RDNA2+, or Intel Arc.

支持Vulkan视频编码的GPU。NVIDIA RTX、AMD RDNA2+或Intel Arc。

Moonlight v6.0.0 or higher. Compatibility with older versions or unofficial ports is not guaranteed.

Moonlight v6.0.0或更高版本。不保证与旧版本或非官方移植版本的兼容性。

**Installation / 安装：**

**Arch / Arch Linux**
The simplest method is to install through the AUR using:
最简单的方法是通过AUR安装：
```bash
yay -S moonshine
```

To run Moonshine for your user:
要为用户运行Moonshine：

Enable user lingering:
启用用户滞留：
```bash
sudo loginctl enable-linger $USER
```
This allows Moonshine to run applications in the user's session even when the user is not logged in. If your user is always logged in when you want to stream, you can skip this step.
这允许Moonshine在用户未登录时也在用户会话中运行应用程序。如果你在想要串流时用户始终已登录，可以跳过此步骤。

Enable the service to start on boot and run immediately:
启用服务以在启动时启动并立即运行：
```bash
sudo systemctl enable --now moonshine@$USER
```

A configuration file is created automatically if the path you provide doesn't exist. When using the AUR package, it defaults to $XDG_CONFIG_HOME/moonshine/config.toml.
如果你提供的路径不存在，会自动创建配置文件。使用AUR软件包时，默认为 $XDG_CONFIG_HOME/moonshine/config.toml。

**Pairing with a client / 与客户端配对：**

When you connect with Moonlight for the first time, it will show a PIN. A notification will appear on the host that you can click to open the pairing page, or you can visit it manually at http://localhost:47989/pin.
当你首次使用Moonlight连接时，它会显示一个PIN码。主机上会出现一个通知，你可以点击它打开配对页面，或者手动访问 http://localhost:47989/pin。

You can also pair from the command line:
你也可以从命令行配对：
```bash
curl -X POST "http://localhost:47989/submit-pin" -d "uniqueid=0123456789ABCDEF&pin=<PIN>"
```

**Adding applications / 添加应用程序：**

Each application runs in its own isolated streaming session. Add them to config.toml like this:
每个应用程序都在其独立的串流会话中运行。像这样将它们添加到 config.toml：
```toml
[[application]]
title = "Steam"
boxart = "/path/to/steam.png" # optional
command = ["/usr/bin/steam", "steam://open/bigpicture"]
```

*   **title:** The name shown in Moonlight. / 在Moonlight中显示的名称。
*   **boxart (optional):** Path to a cover image. / 封面图片的路径。
*   **command:** The command to run. First entry is the executable, the rest are arguments. / 要运行的命令。第一个条目是可执行文件，其余是参数。
*   **pre_command (optional):** Commands to run before launching the application. Each entry is a separate command, executed in order. Runs synchronously — the session waits for all to finish. / 启动应用程序前运行的命令。每个条目是单独的命令，按顺序执行。同步运行——会话会等待所有命令完成。
*   **post_command (optional):** Commands to run after the streaming session ends. Each entry is a separate command, executed in order. Runs synchronously — the server waits for all to finish. / 串流会话结束后运行的命令。每个条目是单独的命令，按顺序执行。同步运行——服务器会等待所有命令完成。

**Example / 示例：**
```toml
[[application]]
title = "Steam"
command = ["/usr/bin/steam", "steam://open/bigpicture"]
pre_command = [
    ["/usr/bin/systemctl", "stop", "conflicting.service"],
    ["/usr/bin/nvidia-smi", "pstate", "50"],
]
post_command = [
    ["/usr/bin/nvidia-smi", "pstate", "performance"],
]
```

**Application scanners / 应用程序扫描器：**

Scanners automatically detect installed applications so you don't have to add them manually.
扫描器会自动检测已安装的应用程序，因此你无需手动添加。

*   **Steam scanner** — finds all installed Steam games:
    **Steam扫描器**——查找所有已安装的Steam游戏：
    ```toml
    [[application_scanner]]
    type = "steam"
    library = "$HOME/.local/share/Steam"
    command = ["/usr/bin/steam", "-bigpicture", "steam://rungameid/{game_id}"]
    ```
*   **Desktop scanner** — finds applications from .desktop files:
    **桌面扫描器**——从.desktop文件查找应用程序：
    ```toml
    [[application_scanner]]
    type = "desktop"
    directories = [
        "$HOME/.local/share/applications",
        "/usr/share/applications",
    ]
    include_terminal = false
    resolve_icons = true
    ```

**FAQ / 常见问题：**

**How does this compare to Sunshine?**
**这与Sunshine相比如何？**
Sunshine supports more platforms and has more features overall. Moonshine is Linux-only. Moonshine runs each streaming session in its own isolated environment, separate from your desktop. This means your host PC stays usable while you stream, and it works without an active desktop session.
Sunshine支持更多平台，并且总体功能更多。Moonshine仅限Linux。Moonshine在独立的隔离环境中运行每个串流会话，与你的桌面分离。这意味着当你串流时，主机PC保持可用，并且它无需活跃的桌面会话即可工作。

**Security / 安全性：**
Moonshine is not designed for use on public networks. The underlying GameStream protocol has limitations that mean traffic is not fully encrypted at the application level. If you need to stream over the internet, use a VPN such as Tailscale, WireGuard, or ZeroTier. Do not expose Moonshine ports directly to the internet.
Moonshine并非为在公共网络上使用而设计。底层GameStream协议有限制，意味着流量在应用层未完全加密。如果你需要通过互联网串流，请使用VPN（如Tailscale、WireGuard或ZeroTier）。切勿将Moonshine端口直接暴露给互联网。
