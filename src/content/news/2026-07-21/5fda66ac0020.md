# Firefox 153 Available With Support For Vulkan Video Decoding, Experimental JPEG-XL  
Firefox 153 发布，支持 Vulkan 视频解码与实验性 JPEG-XL

Written by Michael Larabel in Mozilla on 20 July 2026 at 07:32 AM EDT.  
作者：Michael Larabel，发表于 Mozilla，2026年7月20日 上午7:32 EDT。

Mozilla has uploaded the Firefox 153.0 release binaries today for the newest monthly update to this cross-platform web browser.  
Mozilla 今日已上传 Firefox 153.0 的发布二进制文件，这是该跨平台网页浏览器的最新月度更新。

Making Firefox 153 exciting is that it's the newest Extended Support Release "ESR" as well as introducing Vulkan Video decode support.  
Firefox 153 的亮点在于它是最新的扩展支持版本（ESR），同时引入了 Vulkan 视频解码支持。

Firefox 153 is the newest ESR release designed for enterprises, schools, and other organization for long-term stability.  
Firefox 153 是最新的 ESR 版本，专为追求长期稳定性的企业、学校及其他组织设计。

On the feature side, Firefox 153 introduces Vulkan Video decoding support.  
在功能方面，Firefox 153 引入了 Vulkan 视频解码支持。

Firefox on Linux to date has focused on the Video Acceleration API (VA-API) that works well for some vendors but lacks official support from NVIDIA unless using the independent NVIDIA-VAAPI driver project that layers VA-API on NVDEC.  
迄今为止，Linux 上的 Firefox 主要依赖视频加速 API（VA-API），该 API 对部分厂商效果良好，但 NVIDIA 并未官方支持，除非使用独立的 NVIDIA-VAAPI 驱动项目（在 NVDEC 上叠加 VA-API 层）。

There are also various embedded drivers and more not supporting VA-API.  
此外，许多嵌入式驱动及其他环境也不支持 VA-API。

Now with Firefox there is initial Vulkan Video decode support for that modern, cross-vendor and cross-platform video acceleration interface.  
如今，Firefox 初步支持 Vulkan 视频解码，该接口是现代、跨厂商且跨平台的视频加速方案。

Firefox 153 also brings a number of PDF enhancements, continued enablement around JPEG-XL, HDR video playback on Windows, and more.  
Firefox 153 还带来多项 PDF 增强功能、持续完善的 JPEG-XL 支持、Windows 上的 HDR 视频播放等特性。

Within the Firefox Labs area is where the experimental support for the JPEG-XL image format can be enabled.  
实验性的 JPEG-XL 图像格式支持可在 Firefox Labs 区域中启用。

More details within the beta release notes.  
更多详情请参阅测试版发布说明。

Firefox 153 release binaries can be downloaded from ftp.mozilla.org.  
Firefox 153 的发布二进制文件可从 ftp.mozilla.org 下载。
