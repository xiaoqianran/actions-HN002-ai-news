# Jelly UI: Soft-body physics for native HTML form controls

# Jelly UI：为原生 HTML 表单控件注入软体物理效果

It's okay to be a little jelly. Jelly UI is a dependency-free Web Components library for soft, tactile product interfaces. Real form controls meet soft-body physics, with dark mode, right-to-left support and WCAG AA color tokens built in.  
0 dependencies 40 custom elements 1 script tag WCAG AA Dark mode RTL
