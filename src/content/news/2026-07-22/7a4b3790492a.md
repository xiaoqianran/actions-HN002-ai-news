一个面向CAD、机器人技术和硬件设计代理的技能库  
A skills library for CAD, robotics, and hardware design agents

### CAD 技能  
CAD Skills

CAD Skills 是一个代理技能库，用于从本地项目文件中生成、检查、采购、切片和传递 CAD 及机器人描述工件。  
CAD Skills is a library of agent skills for generating, inspecting, sourcing, slicing, and handing off CAD and robot-description artifacts from local project files.

**🧰 技能**  
🧰 Skills

安装此库可为代理提供专注于 CAD、制造、机器人描述文件、仿真和本地审查的工作流。  
Install the library to give agents focused workflows for CAD, fabrication, robot description files, simulation, and local review.

**技能摘要**  
Skill Summary

**源 CAD**  
Source CAD  
从自然语言或图像请求创建和编辑 CAD 模型，主要输出为 STEP，并可选择导出为 STL、3MF 和 GLB。  
Creates and edits CAD models from plain-language or image requests, with STEP as the main output along with options to export to STL, 3MF and GLB.  
`skills/cad`

**CAD 查看器**  
CAD Viewer  
在本地浏览器中显示 CAD、G-code 和机器人文件的预览。  
Shows local browser previews for CAD, G-code, and robot files.  
`skills/cad-viewer`

**step.parts**  
step.parts  
查找现成的 STEP 零件，如螺丝、轴承、电机和连接器。  
Finds off-the-shelf STEP parts like screws, bearings, motors, and connectors.  
`skills/step-parts`

**DXF**  
DXF  
从 Python 源代码或 CAD 几何体创建 2D DXF 图纸，如轮廓、模板、垫片和切割布局。  
Creates 2D DXF drawings like profiles, templates, gaskets, and cut layouts from Python sources or CAD geometry.  
`skills/dxf`

**URDF**  
URDF  
编写包含链接、关节、限位、惯性和网格的机器人结构文件。  
Writes robot structure files with links, joints, limits, inertials, and meshes.  
`skills/urdf`

**SRDF**  
SRDF  
向 URDF 添加 MoveIt 规划组、末端执行器、姿态和碰撞规则。  
Adds MoveIt planning groups, end effectors, poses, and collision rules to a URDF.  
`skills/srdf`

**SDF**  
SDF  
创建包含框架、物理、传感器和灯光的仿真模型和世界。  
Creates simulator models and worlds with frames, physics, sensors, and lights.  
`skills/sdf`

**SendCutSend**  
SendCutSend  
在上传到 SendCutSend 之前检查 DXF 和 STEP 文件。  
Checks DXF and STEP files before upload to SendCutSend.  
`skills/sendcutsend`

**G-code**  
G-code  
使用真实的切片器 CLI 将受支持的网格文件切片为经过验证的、打印机配置的 FDM .gcode。  
Slices supported mesh files into validated, printer-profiled FDM .gcode with real slicer CLIs.  
`skills/gcode`

**Bambu Labs**  
Bambu Labs  
对经过验证的 .gcode 进行干运行、上传并谨慎启动本地 Bambu Lab 打印作业。  
Dry-runs, uploads, and cautiously starts local Bambu Lab print jobs from validated .gcode.  
`skills/bambu-labs`

**隐式 CAD**  
Implicit CAD  
使用 GLSL 有符号距离场和 CAD 查看器光线行进渲染创建浏览器原生隐式 CAD 模型。实验性。  
Creates browser-native implicit CAD models using GLSL signed-distance fields and CAD Viewer raymarch rendering. Experimental.  
`skills/implicit-cad`

### 💻 安装  
💻 Installation

对于生产用途，请从 main 分支安装或克隆；该分支包含提供商安装程序所需的生成技能/插件输出。  
For production use, install or clone from main; that branch contains the generated skill/plugin outputs needed by provider installers.

**技能安装**  
Skills Install

使用 Skills CLI 安装 CAD Skills：  
Install CAD Skills with the Skills CLI:
```bash
npx skills install earthtojake/text-to-cad
```
这是首选的安装路径。它为支持的代理直接安装各个技能。  
This is the preferred installation path. It installs the individual skills directly for supported agents.

**插件**  
Plugins

也提供提供商原生插件安装，适用于 Codex 和 Claude Code：  
Provider-native plugin installs are also available for Codex and Claude Code:
```bash
# Codex
codex plugin marketplace add earthtojake/text-to-cad
codex plugin add cad@text-to-cad

# Claude Code
claude plugin marketplace add earthtojake/text-to-cad
claude plugin install cad@text-to-cad
```
如果新安装的技能未出现，请重启您的代理。  
Restart your agent if newly installed skills do not appear.

对于本地开发，从 develop 分支创建分支，向 develop 分支发起 PR，并使用 CONTRIBUTING.md 中的符号链接工作流。  
For local development, branch from develop, open PRs against develop, and use the symlink workflow in CONTRIBUTING.md.

### 🧪 基准测试  
🧪 Benchmarks

仓库通过 Git LFS 在 `assets/**` 和 `benchmarks/**` 中存储大型资产，并默认从 LFS 拉取中排除这些树，以便轻量级克隆不获取 GIF 资产。基准测试 markdown 保持普通 Git 以便可读的差异。  
The repo stores heavyweight assets in `assets/**` and `benchmarks/**` through Git LFS and excludes those trees from default LFS pulls so lightweight clones do not fetch GIF assets. Benchmark markdown remains normal Git for readable diffs.

要在本地仅水合基准测试资产，请运行：  
To hydrate only the benchmark assets locally, run:
```bash
git lfs pull --include="benchmarks/**"
```

### 🎯 目标提示输出  
🎯 Target Prompt Output

**1. 带有四个孔的矩形校准块**  
1 Rectangular calibration block with four holes  
创建一个居中的 100 x 60 x 20 毫米块，带有四个 8 毫米垂直通孔。仅在顶部外轮廓添加 2 毫米倒角。  
Create a centered 100 x 60 x 20 mm block with four 8 mm vertical through-holes. Add only a 2 mm chamfer on the top outer perimeter.

**2. 带有螺栓孔图案的圆形法兰**  
2 Circular flange with bolt-hole pattern  
创建一个直径 80 毫米、厚 10 毫米的圆形法兰，带有一个 30 毫米中央通孔。在 60 毫米螺栓圆上添加六个 6 毫米通孔，并对外部圆形边缘进行圆角处理。  
Create an 80 mm diameter, 10 mm thick circular flange with a 30 mm central through-bore. Add six 6 mm through-holes on a 60 mm bolt circle and fillet the outside circular edges.

**3. 带有支撑和两个孔方向的 L 型支架**  
3 L-bracket with gussets and two hole directions  
从底板和后垂直板创建一个 L 型支架。添加垂直底板孔、水平背板孔、两个三角形支撑，以及圆角的底板/背板过渡。  
Create an L-bracket from a base plate and rear vertical plate. Add vertical base holes, horizontal back-plate holes, two triangular gussets, and a filleted base/back transition.

**4. 带键槽的阶梯轴**  
4 Stepped shaft with keyway  
沿 X 轴创建一根 120 毫米轴，带有 20/30/20 毫米直径的阶梯段。添加端部倒角，并在中间段顶部添加一个浅矩形键槽。  
Create a 120 mm shaft along X with 20/30/20 mm diameter stepped sections. Add end chamfers and a shallow rectangular keyway on top of the middle section.

**5. 带凸台的开放式顶部电子外壳**  
5 Open-top electronics enclosure with bosses  
创建一个中空开放式顶部外壳，壁厚和地板为 3 毫米。添加四个内部支架，带中心盲孔，以及外部垂直角部 2 毫米圆角。  
Create a hollow open-top enclosure with 3 mm walls and floor. Add four internal standoffs with centered blind holes and 2 mm outside vertical corner fillets.

**6. 具有减轻重量切口航空航天式耳轴支架**  
6 Aerospace-style clevis bracket with lightening cutouts  
创建一个对称的耳轴支架，带有底板、两个圆凸耳、底板安装孔和水平凸耳孔。添加三角形减轻重量切口、加强肋和圆角过渡。  
Create a symmetric clevis bracket with a base plate, two rounded lugs, base mounting holes, and a horizontal lug bore. Add triangular lightening cutouts, reinforcing ribs, and rounded transitions.

**7. 具有冷却鳍的径向发动机式气缸**  
7 Radial-engine-style cylinder with cooling fins  
创建一个垂直发动机气缸形式，带有中心桶、12 个冷却鳍、底板法兰和顶部盖。添加一个 35 度倾斜的火花塞凸台，带有同轴通孔。  
Create a vertical engine-cylinder form with a central barrel, 12 cooling fins, a base flange, and a top cap. Add a 35 degree angled spark-plug boss with a coaxial through-hole.

**8. 具有后弯叶片的离心式叶轮**  
8 Centrifugal impeller with backward-curved blades  
创建一个离心式叶轮，带有背板、轮毂和通孔。添加 12 个融合的后弯叶片，从根部到尖端 sweep 约 45 度。  
Create a centrifugal impeller with a backplate, hub, and through-bore. Add 12 fused backward-curved blades sweeping about 45 degrees from root to tip.

**9. 带有螺旋栏杆的螺旋楼梯**  
9 Spiral staircase with helical handrail  
创建一个微型螺旋楼梯，带有中心柱、基盘和 20 个上升楔形踏板。添加一个单圈螺旋栏杆，以及踏板外端的垂直栏杆。  
Create a miniature spiral staircase with a central column, base disk, and 20 rising wedge treads. Add a one-revolution helical handrail and vertical balusters at the tread outer ends.

**10. 简化行星齿轮级**  
10 Simplified planetary gear stage  
创建一个平面行星齿轮组件，带有独立的太阳轮、行星轮、齿圈、行星架和销体。使用简化的梯形齿，在 42 毫米半径圆上围绕太阳轮放置三个行星轮。  
Create a flat planetary gear assembly with separate sun, planet, ring, carrier, and pin bodies. Use simplified trapezoidal teeth and place three planets around the sun on a 42 mm radius circle.
