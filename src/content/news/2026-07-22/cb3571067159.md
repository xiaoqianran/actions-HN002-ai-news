**Title:** ForensicNet: Lightweight Attention-Enhanced MobileNetV2 for Automated Face Identification  
**标题：** ForensicNet：用于自动人脸识别的轻量级注意力增强MobileNetV2  

**Authors:** Savitha N J, Lata B T  
**作者：** Savitha N J, Lata B T  

**Abstract:** In forensic environments, automated identification of perpetrators is difficult due to pose changes, changes in light, occlusion, and lack of labeled data.  
**摘要：** 在法医环境中，由于姿态变化、光照变化、遮挡和缺乏标注数据，对犯罪者的自动识别具有挑战性。  

This paper presents ForensicNet, a lightweight deep learning framework for forensic face recognition that enhances attention.  
本文提出了ForensicNet，一个用于法医人脸识别的轻量级深度学习框架，该框架增强了注意力机制。  

The suggested model combines the MobileNetV2 backbone with Convolutional Block Attention Modules (CBAM) to improve the learning of discriminative features while maintaining computational speed.  
所提出的模型将MobileNetV2骨干网络与卷积块注意力模块（CBAM）相结合，以提升判别性特征的学习能力，同时保持计算速度。  

A two-phase transfer learning strategy with adaptive layer unfreezing is used to improve domain adaptation and reduce overfitting.  
采用带有自适应层解冻的两阶段迁移学习策略，以改善领域适应并减少过拟合。  

This study used publicly available datasets such as LFW and SCFace, with 15,000 facial images spanning 68 identity classes.  
本研究使用了LFW和SCFace等公开数据集，包含涵盖68个身份类别的15,000张人脸图像。  

The proposed model outperforms baseline architectures such as AlexNet, ResNet-50, and MobileNetV2, with an accuracy of 92.4%, a precision of 90.8%, and a recall of 89.5%.  
所提出的模型在准确率（92.4%）、精确率（90.8%）和召回率（89.5%）上均优于AlexNet、ResNet-50和MobileNetV2等基线架构。  

Additionally, the framework requires only 2.1 GFLOPs per inference, and hence can be used in real-time forensic surveillance applications.  
此外，该框架每次推理仅需2.1 GFLOPs，因此可用于实时法医监控应用。
