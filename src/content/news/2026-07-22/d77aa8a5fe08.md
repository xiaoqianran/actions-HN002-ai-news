# Jack Dorsey is taking on Slack with Buzz, a group chat platform for teams and their AI agents  
# 杰克·多西携 Buzz 挑战 Slack：一个供团队及其 AI 智能体使用的群聊平台

Twitter and Block co-founder Jack Dorsey announced a new app on Tuesday called Buzz. Positioned as a challenger to Slack and GitHub, Buzz is a group chat platform for the workplace that puts humans and their AI agents in the same conversations.  
Twitter 和 Block 的联合创始人杰克·多西于周二宣布了一款名为 Buzz 的新应用。Buzz 被定位为 Slack 和 GitHub 的挑战者，是一个面向工作场所的群聊平台，将人类及其 AI 智能体置于同一对话中。

Dorsey wrote on X that Buzz is “model-agnostic, decentralized, self-sovereign, and open source.” This product seems to be more than just a Dorsey passion project. According to its website, Buzz was built by Dorsey’s company Block, which also operates products like Square, Cash App, Afterpay, and Tidal.  
多西在 X 上写道，Buzz 是“与模型无关、去中心化、自主主权且开源的”。这款产品似乎不仅仅是多西的个人兴趣项目。根据其官网信息，Buzz 由多西的公司 Block 开发，该公司还运营 Square、Cash App、Afterpay 和 Tidal 等产品。

“我们推出 BUZZ！一个面向各种规模团队和智能体的新群聊平台，旨在减少我们对 Slack 和 GitHub 的依赖。与模型无关、去中心化、自主主权且开源。🐝” —— 杰克 (@jack) 2026年7月21日

As startups increasingly rely on AI agents to get work done, it can be challenging for employees to collaborate on various tasks across different platforms. Buzz’s utility is that it merges several different workflows into one workspace.  
随着初创企业越来越依赖 AI 智能体来完成工作，员工在不同平台上协作处理各种任务可能变得具有挑战性。Buzz 的实用性在于它将多种不同的工作流合并到一个工作空间中。

It looks a lot like Slack, but with native AI agents and the ability to manage GitHub projects all from the same window. Since the platform is open source, developers can make their own Buzz instance feel more customized to the needs and workflows of their specific team.  
它看起来很像 Slack，但内置了 AI 智能体，并且能够从同一个窗口管理 GitHub 项目。由于该平台是开源的，开发者可以让他们自己的 Buzz 实例更贴合特定团队的需求和工作流。

If a team needs a new feature, they can build it and deploy it on their own, since they have full access to the source code.  
如果团队需要新功能，他们可以自行构建并部署，因为他们可以完全访问源代码。

Dorsey isn’t the only entrepreneur trying to pursue AI-native alternatives or additions to Slack. Paradigm partner and CTO Georgios Konstantopoulos recently unveiled a similar open source product called Centaur, which he describes as a “virtual employee” that runs either inside of Slack or via an API.  
多西并非唯一一位试图开发 Slack 的 AI 原生替代方案或附加功能的创业者。Paradigm 合伙人兼首席技术官 Georgios Konstantopoulos 最近推出了一款名为 Centaur 的类似开源产品，他将其描述为一个“虚拟员工”，可以在 Slack 内部或通过 API 运行。

“There’s a lot of room for improvement for agents that live in Slack and can do more work than just coding for teams. In the enterprise setting, this means that you’ll want to self-host for security and control, and you want people to use it in Slack,” Konstantopoulos wrote on X.  
Konstantopoulos 在 X 上写道：“对于驻留在 Slack 中并能做比仅为团队编码更多工作的智能体，还有很多改进空间。在企业环境中，这意味着你需要自托管以确保安全和控制，并且希望人们在 Slack 中使用它。”

For newer startups that are using AI agents and don’t have an established presence on Slack, Buzz (or its competitors) could be worth investigating. But Buzz itself admits that it is in its “early stages,” so it’s probably not a good idea to port your team over just yet.  
对于使用 AI 智能体且尚未在 Slack 上建立稳固存在的新兴初创公司来说，Buzz（或其竞争对手）值得研究。但 Buzz 自身承认它仍处于“早期阶段”，因此现在可能还不是将团队迁移过去的好时机。

Buzz’s free desktop app is available now for macOS, Windows, and Linux, and the code for the app has been uploaded to GitHub.  
Buzz 的免费桌面应用现已适用于 macOS、Windows 和 Linux，该应用的代码已上传至 GitHub。
