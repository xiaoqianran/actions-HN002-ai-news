# Incremental – A library for incremental computations

# Incremental – 一个用于增量计算的库

Incremental is a library that gives you a way of building complex computations that can update efficiently in response to their inputs changing, inspired by the work of Umut Acar et. al. on self-adjusting computations.

Incremental 是一个库，它提供了一种构建复杂计算的方法，这些计算可以高效地响应其输入的变化而更新，其灵感来源于 Umut Acar 等人关于自适应计算的工作。

Incremental can be useful in a number of applications, including: Building large calculations (of the kind you might build into a spreadsheet) that can react efficiently to changing data. Constructing views in GUI applications that can incorporate new data efficiently. Computing derived data while guaranteeing that the derived data stays in sync with the source data, for instance filtering or inversing a mapping.

Incremental 在多种应用中都非常有用，包括：构建大型计算（例如你可能在电子表格中构建的那种），这些计算可以高效地响应变化的数据。在图形用户界面（GUI）应用程序中构建视图，这些视图可以高效地整合新数据。计算派生数据，同时保证派生数据与源数据保持同步，例如过滤或反转映射关系。

You can find detailed documentation of the library and how to use it in `incremental/src/incremental_intf.ml`. You can also find an informal introduction to the library in this blog post and this video.

你可以在 `incremental/src/incremental_intf.ml` 中找到关于该库及其使用方法的详细文档。你也可以在这篇博客文章和这个视频中找到对该库的非正式介绍。
