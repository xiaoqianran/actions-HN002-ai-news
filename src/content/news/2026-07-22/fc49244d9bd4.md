### 1. Building GPU-native workflows with the cuDF API  
### 1. 使用cuDF API构建GPU原生工作流

If you’re starting a new project and want to work directly with GPU DataFrames, cuDF could be a great option.  
如果你正在启动新项目并希望直接使用GPU DataFrame，cuDF可能是一个绝佳选择。

It is part of RAPIDS, NVIDIA’s open source suite of GPU-accelerated libraries for data science and analytics.  
它是RAPIDS的一部分，NVIDIA用于数据科学和分析的GPU加速开源库套件。

cuDF is a Python GPU DataFrame library for manipulating tabular data using a pandas-like API.  
cuDF是一个Python GPU DataFrame库，使用类似pandas的API操作表格数据。

So, if you’re familiar with pandas, much of the syntax will look familiar.  
因此，如果你熟悉pandas，大部分语法会看起来很熟悉。

The difference, however, is that unlike pandas, cuDF executes these operations on the GPU rather than the CPU.  
但区别在于，与pandas不同，cuDF在GPU而非CPU上执行这些操作。

Something to note, however, is that while cuDF closely follows the pandas API, it is not a complete drop-in replacement, so some behaviors may differ.  
需要注意的是，虽然cuDF closely follows pandas API，但它并非完全即插即用的替代品，某些行为可能有所不同。

#### Loading data into a cuDF DataFrame  
#### 将数据加载到cuDF DataFrame

Loading data into a cuDF DataFrame works just like pandas.  
将数据加载到cuDF DataFrame与pandas类似。

In most cases, you simply replace `pandas.read_*` with the corresponding `cudf.read_*` function.  
在大多数情况下，你只需将`pandas.read_*`替换为相应的`cudf.read_*`函数。

Here’s how you’d load a Parquet file using cuDF:  
以下是使用cuDF加载Parquet文件的方法：

```python
import cudf

taxi_url = 'https://d37ci6vzurychx.cloudfront.net/trip-data/yellow_tripdata_2023-01.parquet'
df_gpu = cudf.read_parquet(taxi_url)
df_gpu.info()
```

Once the data is loaded into GPU memory, subsequent DataFrame operations run on the GPU.  
一旦数据加载到GPU内存中，后续的DataFrame操作将在GPU上运行。

#### Moving between pandas and cuDF  
#### 在pandas和cuDF之间转换

Many data science workflows use libraries that only support pandas.  
许多数据科学工作流使用仅支持pandas的库。

You can easily move data between CPU and GPU DataFrames using cuDF, when required.  
在需要时，你可以使用cuDF在CPU和GPU DataFrame之间轻松移动数据。

**Converting a pandas DataFrame to cuDF:**  
**将pandas DataFrame转换为cuDF：**

```python
df_gpu = cudf.from_pandas(df_cpu)
print(type(df_gpu))
# ---------------------------------------
# <class 'cudf.core.dataframe.DataFrame'>
```

**Converting a cuDF DataFrame back to pandas:**  
**将cuDF DataFrame转换回pandas：**

```python
df_cpu = df_gpu.to_pandas()
print(type(df_cpu))
# ---------------------------------------
# <class 'pandas.core.frame.DataFrame'>
```

These conversions are useful when a downstream library only supports pandas or when you want to migrate an existing workflow to the GPU.  
当下游库仅支持pandas，或你希望将现有工作流迁移到GPU时，这些转换非常有用。
