# An ESP32 and Eight ToF Sensors Give This Antique Lamp a Sixth Sense  
# ESP32与八个ToF传感器为这盏古董灯赋予第六感

What if a floor lamp could feel you crossing the room and answer with light and sound? That is the trick behind Bloom Chandelier, an interactive installation built around a brass floor lamp handed down from the maker's great-grandmother.  
如果一盏落地灯能感知你穿越房间，并以光与声回应，那会怎样？这就是“Bloom Chandelier”互动装置的奥秘——它围绕一盏从制作者曾祖母那里传承下来的黄铜落地灯打造。

Instead of a switch, the lamp watches the space around it. Walk closer and it glows brighter while soft tones swell up from nowhere. Step away and both the light and the sound fade back into silence, leaving a short trail behind you like a wake.  
它没有开关，而是默默注视周围空间。你越走越近，灯光便愈发明亮，柔和的音调亦凭空渐强；你转身离去，光与声便重归寂静，只在身后留下一道短暂的余晖，如航迹般消散。

How does it know you are there? Eight VL53L1X time-of-flight sensors ring the lamp's collar, giving it a full 360-degree view that reaches roughly four feet out.  
它如何知晓你的存在？八颗VL53L1X飞行时间传感器环绕灯颈，赋予其约四英尺（1.2米）半径的完整360度视野。

These are not motion detectors. Each one measures actual distance many times a second, so the firmware can react to how near you are rather than just whether you moved.  
它们并非运动探测器。每颗传感器每秒多次测量实际距离，使固件能根据你的接近程度而非仅仅移动状态作出反应。

The catch: every VL53L1X ships with the same default I2C address, so eight of them on one bus would collide instantly.  
难点在于：每颗VL53L1X出厂时均使用相同的默认I2C地址，若八颗挂接在同一总线上会立即冲突。

The wiring answer: An Adafruit HUZZAH32 ESP32 Feather runs the show, talking to the sensors through a PCA9548A eight-channel I2C multiplexer.  
解决方案：一台Adafruit HUZZAH32 ESP32 Feather主板掌控全局，通过PCA9548A八通道I2C多路复用器与传感器通信。

The code polls each sensor one at a time in a round-robin loop, switching multiplexer channels so no two sensors ever share the bus at once.  
代码以轮询循环逐一读取各传感器，通过切换多路复用器通道，确保任意时刻仅有一个传感器占用总线。

Aiming each sensor outward also keeps their infrared beams from bleeding into each other.  
同时将每颗传感器向外瞄准，避免其红外光束相互干扰。

Those distance readings drive a 59-pixel warm-white LED strip split into eight zones, one per sensor, with brightness following a squared proximity curve so the glow ramps up fast as you approach.  
这些距离数据驱动一条59像素的暖白色LED灯带，它被划分为八个区域（每传感器对应一个），亮度按距离平方曲线变化——你越靠近，光芒骤增。

The same numbers stream over USB serial to a Raspberry Pi 4 Model B, where a Python engine maps each sensor to one note of a two-octave C major pentatonic scale.  
相同数据通过USB串口传输至树莓派4 Model B，其上的Python引擎将每个传感器映射到一个两个八度内的C大调五声音阶音符。

The notes are not synth patches but recordings of fingers circling wine-glass rims, pitch-shifted in SciPy from four sampled anchors.  
这些音符并非合成音色，而是手指沿酒杯杯口划动的录音，通过SciPy基于四个采样锚点进行变调处理。

Try a smaller version: You do not need eight sensors to learn the core ideas here. Wire a single VL53L1X to an ESP32 over I2C, read the distance, and map it to the brightness of one LED.  
尝试缩小版：无需八颗传感器即可掌握核心原理。将单颗VL53L1X通过I2C连接至ESP32，读取距离并将其映射到一颗LED的亮度。

Once that works, add a PCA9548A and a second sensor to practice channel switching, the exact skill that lets Bloom Chandelier scale to eight.  
一旦实现，再加入PCA9548A和第二颗传感器以练习通道切换——这正是让Bloom Chandelier扩展至八颗传感器的关键技能。
