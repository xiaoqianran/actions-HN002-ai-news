# iOS code could reportedly let Apple cut off apps when users miss iPhone payments

Code found in an iOS 27 beta would allow Apple to put a financed iPhone in “Restricted Mode” if it detects any missed payments, 9to5Mac reports.  
9to5Mac报道称，在iOS 27测试版中发现的代码允许苹果在检测到任何未付款项时，将分期付款的iPhone置于“受限模式”。

The finding follows a story from Bloomberg earlier this week claiming Apple is preparing to launch a new “Apple Upgrade” financing program for leasing new devices.  
这一发现紧随本周早些时候彭博社的一篇报道，该报道称苹果正准备推出一个新的“Apple Upgrade”融资计划，用于租赁新设备。

iPhones in “Restricted Mode” would reportedly lose access to nearly all apps, except for basics like the Phone app, Settings, and the App Store.  
据报道，处于“受限模式”的iPhone将无法访问几乎所有应用，除了电话、设置和应用商店等基础功能。

It uses a new “App Managed Features” system that allows financing or partner apps to continually check if a financed device is in good standing.  
它使用一个新的“应用管理功能”系统，允许融资或合作伙伴应用持续检查分期设备是否状态良好。

According to 9to5Mac, the code also includes a new “Partner Finance Lock” intended to block users from “erasing, reselling, or stripping a restricted device for parts.”  
据9to5Mac称，该代码还包括一个新的“合作伙伴金融锁”，旨在阻止用户“擦除、转售或拆解受限设备以获取零件”。

While the feature is part of Find My, it reportedly doesn’t give financing partners access to the phone’s location.  
虽然该功能是“查找”网络的一部分，但据报道它不会让融资合作伙伴访问手机的位置。

This is from a beta version of iOS 27, so it’s possible this code and the new “Restricted Mode” could get changed or removed before Apple officially releases iOS 27.  
这来自iOS 27的测试版，因此这些代码和新的“受限模式”在苹果正式发布iOS 27之前可能会被更改或移除。

The “Apple Upgrade” program hasn’t been officially announced yet, either.  
“Apple Upgrade”计划也尚未正式宣布。

Apple did not immediately respond to a request for comment.  
苹果未立即回复置评请求。
