# Pumpkin-MC / Pumpkin

Pumpkin is a Minecraft server built entirely in Rust, offering a fast, efficient, and customizable experience. It prioritizes performance and player enjoyment while adhering to the core mechanics of the game.

Pumpkin 是一个完全使用 Rust 构建的 Minecraft 服务器，提供快速、高效且可定制的体验。它在坚持游戏核心机制的同时，优先考虑性能和玩家体验。

## Goals

**Performance:** Leveraging multi-threading for maximum speed and efficiency.  
**性能：** 利用多线程实现最大速度和效率。

**Compatibility:** Supports the latest Java & Bedrock Minecraft server version while adhering to Vanilla game mechanics.  
**兼容性：** 支持最新的 Java 和基岩版 Minecraft 服务器版本，同时遵循原版游戏机制。

**Security:** Prioritizes security by preventing known security exploits.  
**安全性：** 通过防止已知的安全漏洞来优先保障安全。

**Flexibility:** Highly configurable, with the ability to disable unnecessary features.  
**灵活性：** 高度可配置，能够禁用不必要的功能。

**Extensibility:** Provides a foundation for plugin development.  
**可扩展性：** 为插件开发提供基础。

## Important

Pumpkin is currently under heavy development. See [what needs to be done before the 1.0.0 Release](https://github.com/pumpkinmc/pumpkin/issues/1).

Pumpkin 目前正处于积极开发阶段。查看[在 1.0.0 版本发布前需要完成的事项](https://github.com/pumpkinmc/pumpkin/issues/1)。
