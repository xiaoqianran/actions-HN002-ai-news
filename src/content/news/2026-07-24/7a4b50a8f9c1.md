# Swenest

Launching today, Swenest is a platform that teaches how to navigate real-world code.

今日上线，Swenest 是一个教授如何驾驭真实世界代码的平台。

It stands out with real merged GitHub PRs as tasks, not artificial problems or symptom-only descriptions; users find the cause themselves. It features progressive AI hints that guide without spoiling, an AI judge that grades against the actual merged fix, and XP-gated difficulty progression.

其独特之处在于：使用真实合并的 GitHub PR 作为任务，而非人工编造的问题或仅描述症状的题目；用户需自行定位问题根源。功能包括：渐进式 AI 提示（引导而不剧透）、基于实际合并修复进行评分的 AI 裁判，以及通过经验值（XP）解锁难度的进阶机制。

Most platforms don't train navigating unfamiliar, real-world codebases with structured guidance within a gamified platform.

大多数平台并未在游戏化环境中，提供结构化指导来训练驾驭陌生真实代码库的能力。
